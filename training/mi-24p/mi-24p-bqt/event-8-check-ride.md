## Check Ride Overview

The Check Ride is the culminating evaluation of the Mi-24P Hind Basic Qualification Training course. It is a **full-profile flight** conducted from cold-and-dark startup to cold-and-dark shutdown. The student is evaluated on all core competencies introduced across Events 1–7.

**Duration:** 60–90 minutes (including ground evaluation)

**Evaluator:** Instructor or qualified check airman — someone who has not been the student's primary instructor is preferred.

**Grading:** Pass / Fail (No Incomplete Items carried into the check ride — all IIs must be resolved before this event)

> **No Autostart.** The student must demonstrate the complete manual startup procedure. Autostart is not permitted during the check ride.

> **No Notes.** The student may not reference written checklists, notes, or guides during the ground evaluation boldface recitation. During the flight, the cockpit checklist is permitted for non-boldface items.

## Prerequisites

Before the check ride begins, the following must be confirmed:

| Item | Requirement |
|---|---|
| **All Events 1–7** | PASS grade (no REPEAT or DOWN pending) |
| **Incomplete Items** | Zero unresolved IIs |
| **Boldface** | Recited at full standard within the last 5 training sessions |
| **Autorotation** | Demonstrated to completion standard in Event 6 |
| **Aircraft configuration** | Training weight (~10,000 kg), no weapons load |
| **Weather** | Day VMC, winds ≤ 8 m/s (crosswind component ≤ 8 m/s) |

## Ground Evaluation

The ground evaluation occurs before the student touches the aircraft.

### Part A — Boldface Recitation

The evaluator calls out the emergency title. The student recites the complete boldface procedure verbatim and in sequence.

| Emergency | Pass Criterion |
|---|---|
| Engine failure in flight (single) | All 6 steps, verbatim, in order |
| Dual engine failure / autorotation | All 7 steps, verbatim, in order |
| Engine fire | All 4 steps, verbatim, in order |
| Ground resonance | Both sub-cases (Nr ≥ 90% and Nr < 90%), all steps |
| Tail rotor failure | All 4 steps, verbatim, in order |

**Fail criterion:** Any single boldface item recited with an omitted step, wrong sequence, or significant deviation from the verbatim text.

> A failed boldface recitation **terminates the check ride**. The student must re-study and attempt the boldface on a subsequent session before the flight portion can proceed.

### Part B — Oral Knowledge Questions

The evaluator will ask 8–10 questions from the following topic areas. The student must answer correctly or self-correct within one follow-up prompt.

| Topic Area | Example Questions |
|---|---|
| Engine limits | "What is the EGT limit during engine start?" / "What is the max continuous torque per engine?" |
| Rotor system | "What is the normal Nr range?" / "What type of rotor system does the Mi-24P use?" |
| Aerodynamics | "What is ETL and how does it affect the Mi-24P on takeoff?" / "What causes LTE and when is it most likely?" |
| Emergency recognition | "What are the indications of ground resonance?" / "How do you identify which engine has failed?" |
| Limitations | "What is the crosswind limit?" / "What is the VNE?" |
| Systems | "What does the backup hydraulic system power?" / "How long do the batteries last on dual generator failure?" |
| Navigation | "What does the DISS-15 provide?" / "How do you track inbound on the ARK-15?" |
| Takeoff & landing | "Why is the running takeoff preferred over the hover takeoff?" / "What is the hover power check decision rule?" |

**Pass criterion:** 80% correct (minimum 7 of 10 questions) with no unsafe knowledge gaps.

## Flight Evaluation Profile

The check ride flight follows this sequence. The student conducts the entire flight — the evaluator observes, records grades, and may introduce emergency scenarios.

### Phase 1 — Startup & Ground Operations

**Graded items:**
- Complete cold-dark startup in correct sequence (APU → Left engine → Right engine → generators → avionics → rotor)
- No hot start (EGT < 900°C throughout)
- Both generators connected before avionics powered
- WSO cockpit configured (weapons SAFE, SPO-15 ON, UV-26 ON)
- Pre-takeoff check completed in order
- Rotor brake released correctly; Nr stabilizes at 95%

### Phase 2 — Hover Operations

**Graded items:**
- Hover power check performed and result stated
- Stationary hover: ±1 m altitude, ±3 m drift, 30 seconds minimum
- At least one pedal turn: ±5° of target heading, altitude maintained
- Hover taxi to runway: altitude ±1 m; speed ≤ 10 km/h

### Phase 3 — Takeoff and Climb

**Graded items:**
- Running takeoff: smooth acceleration; airborne before 200 m; no bouncing
- ETL transition: smooth (no large attitude excursions)
- Gear retraction: after positive rate confirmed
- Climb to assigned altitude: ±50 m of assigned; Vy (120–140 km/h) ±10 km/h

### Phase 4 — Basic Airwork

The evaluator assigns maneuvers from the following list. Student executes without prompting beyond the initial brief.

| Maneuver | Standard |
|---|---|
| Level flight at 200 km/h | ±50 m altitude, ±10 km/h airspeed |
| 30° banked turn (both directions) | Ball centered; ±50 m altitude |
| 45° banked turn (one direction) | Ball centered; ±50 m altitude; no G exceedance |
| Climb — Vy (best rate) | ±10 km/h of 120–140 km/h; positive rate throughout |
| Descent — cruise | Controlled; no settling with power indication |
| Slow flight below ETL | Controlled flight at 30 km/h; no loss of control |
| Force trim demonstration | Correct technique; aircraft flies hands-off when trimmed |
| Autopilot engagement/heading change | All 4 channels; heading change with AP on; smooth disengagement |

### Phase 5 — Navigation Leg

The evaluator assigns a 2-leg navigation task:
- **Leg 1:** Navigate to a designated point using ARK-15 or RSBN
- **Leg 2:** Return to the airfield using a different nav system

| Standard | Requirement |
|---|---|
| System use | Correct system setup and tracking technique |
| Wind correction | Demonstrates correction when needle drifts |
| Altitude | ±50 m during transit |
| Airspeed | ±10 km/h of assigned cruise speed |
| Position accuracy | Arrives within ±3 km of intended waypoint |

### Phase 6 — Emergency Procedures

The evaluator will introduce at least **2 emergency scenarios** during the flight. The student does not know when or which emergencies will be introduced.

| Possible Scenarios | Method |
|---|---|
| Simulated single-engine failure | Evaluator announces verbally; student executes boldface |
| Simulated hydraulic failure | Evaluator announces; student changes control technique |
| Simulated electrical failure | Evaluator announces; student reduces loads, plans landing |
| Practice autorotation | Standard procedure; power recovery at 10 m AGL |
| LTE recognition | Evaluator sets up wind conditions; student responds |

**Pass criterion for each emergency:**
- Boldface steps executed correctly and in order
- No unsafe aircraft attitude or Nr exceedance during execution
- Correct airspeed maintained

### Phase 7 — Traffic Pattern and Landings

The student will fly at least **3 traffic patterns** during the check ride.

| Pattern | Type | Special Instruction |
|---|---|---|
| 1 | Running landing | Standard; evaluator observes pattern discipline |
| 2 | Go-around (evaluator calls) | Must execute clean go-around; re-pattern; land |
| 3 | Running or hover landing | Evaluator's choice based on power margin |

**Optional (evaluator's discretion):** Steep approach or crosswind demonstration if conditions allow.

| Standard | Requirement |
|---|---|
| Pattern altitude | 200 m ±25 m |
| Gear extension | Abeam threshold; 3 green lights verbally confirmed |
| Final approach speed | 120–140 km/h → decelerating to 80–100 km/h |
| Running landing | Main gear first; centerline ±5 m; no skip |
| Go-around | Positive climb within 5 sec; clean execution; re-patterns correctly |

### Phase 8 — Shutdown

Full shutdown sequence from landing to cold-dark. Evaluated from memory.

| Standard | Requirement |
|---|---|
| Sequence | Avionics → cool-down (2 min) → right engine → left engine → rotor brake → generators → batteries |
| Rotor brake | Applied after Nr < 50% only |
| WSO cockpit | Weapons SAFE; all systems OFF |
| Cold-dark | Both cockpits fully dark and cold at completion |

## Grading Standards

### Overall Result

| Result | Definition |
|---|---|
| **PASS — Basic Qualified** | All standards met; student is Mi-24P BQT qualified |
| **PASS WITH COMMENT** | All standards met with minor deficiencies noted for follow-on training |
| **FAIL — Repeat Required** | One or more phase standards not met; specific areas identified for remediation; re-check after practice |
| **DOWN** | Unsafe act observed by evaluator; immediate debrief; additional review before re-check eligibility |

### Automatic Fail Conditions

Any one of the following constitutes an immediate check ride failure:

- [ ] Any boldface step omitted or executed in wrong order (ground or in-flight)
- [ ] EGT exceeds 900°C during engine start (uncontrolled hot start)
- [ ] Nr exceedance during autorotation (Nr > 105%)
- [ ] Bank angle exceeds 60° during any maneuver
- [ ] G-loading exceeds +2.5 G or goes below −0.5 G
- [ ] Gear-up landing (landing with gear retracted)
- [ ] Fuel below 300 kg (bingo) without immediate action and declaration
- [ ] Unsafe act that would endanger personnel in real-world operations
