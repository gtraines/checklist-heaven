## Learning Objectives

Upon completion of this event the student will be able to:
- [ ] Fly a standard VFR traffic pattern at the correct altitudes and airspeeds
- [ ] Execute a running landing (the Mi-24P standard arrival)
- [ ] Execute a hover landing when directed and power permits
- [ ] Perform a steep approach for obstacle-rich environments
- [ ] Execute a safe go-around from a destabilized approach
- [ ] Demonstrate crosswind approach and landing technique
- [ ] Manage landing gear through the pattern (extension, verification, emergency awareness)
- [ ] Perform a complete shutdown sequence — both cockpits, cold-and-dark — after landing

## Ground Brief

### Landing Is Where the Mi-24P Punishes Complacency

More accidents in helicopter training occur during approach and landing than any other phase. For the Mi-24P specifically:

- **Running landing is the standard** — not hover landing. Less power, more control, less risk.
- **Hovering on final is an invitation to settle with power** — maintain airspeed until committed.
- **Go-around commitment point is 50 m AGL** — below this, commit to the landing.
- **Power check before any hover landing** — if torque > 85% to hover, do a running landing instead.

### The Traffic Pattern

The Mi-24P VFR traffic pattern uses **200 m AGL** as the standard pattern altitude (Soviet/Russian convention). The altimeter should be set to **QFE** (reads zero at field elevation) so the student reads altitude above the airfield directly.

```
           CROSSWIND
              ↑
UPWIND        │         DOWNWIND
→→→→→→→→→→→→→┼→→→→→→→→→→→→→
              │         (200 m, 200–220 km/h)
              │         (GEAR DOWN abeam runway end)
              │
              └────────────────────
                                   ↓
              ┌────────────────────┘
              │   BASE (150 m, 160–180 km/h)
              │
              ▼
           FINAL (Decelerating to 120–140 km/h → 80–100 km/h short final)
              │
              ▼
         TOUCHDOWN / GO-AROUND POINT (50 m = commit point)
```

### Downwind Checklist (Before turning base)

| Item | Action | Standard |
|---|---|---|
| **Landing Gear** | DOWN | 3 green lights confirmed |
| **Airspeed** | Decelerating < 200 km/h | |
| **Radar Altimeter** | Bug set to 50 m | Decision height warning |
| **Fuel** | Checked — sufficient for approach + go-around | > 300 kg |
| **Autopilot** | OFF | Hand-fly the approach |
| **Weapons** | SAFE | Verbally confirmed |

### Running Landing — Key Parameters

| Phase | Speed | Notes |
|---|---|---|
| Final approach | 120–140 km/h | Decelerating, 3–5° glidepath |
| Short final | 80–100 km/h | Continuing deceleration |
| Touchdown (ground speed) | 50–70 km/h | Main gear first |
| Rollout | 50 → 0 km/h | Controlled braking |

## In-Sim Procedures

### Exercise 1 — First Traffic Pattern and Running Landing

**Setup:** Start from runway (after startup and pre-takeoff check from Event 2 procedure). Fly a single traffic pattern.

#### Upwind / Crosswind

| Step | Action | Notes |
|---|---|---|
| 1 | Running takeoff — airborne and climbing | From Event 3 procedure |
| 2 | Gear UP — positive rate confirmed | `G` key or cockpit handle |
| 3 | Climb at 120–140 km/h (Vy) | Climb to 200 m |
| 4 | At 200 m — level off; turn crosswind | 90° turn to crosswind leg |

#### Downwind

| Step | Action | Notes |
|---|---|---|
| 5 | Turn to parallel the runway (downwind heading) | |
| 6 | Cruise at 200–220 km/h, 200 m altitude | |
| 7 | **Abeam the runway threshold** — Landing Gear DOWN | `G` — verify 3 green lights |
| 8 | Decelerate to 200 km/h | Begin slowing |
| 9 | Complete downwind checklist | See table above |
| 10 | Turn base | When runway end is 45° behind |

#### Base

| Step | Action | Notes |
|---|---|---|
| 11 | Descend at 3–5 m/s | Reduce collective |
| 12 | Decelerate to 160–180 km/h | |
| 13 | At 150 m — stable on base | |
| 14 | Turn final | When runway is ahead, clear of any conflict |

#### Final Approach

| Step | Action | Notes |
|---|---|---|
| 15 | Roll out on runway centerline | |
| 16 | Continue decelerating: 140 → 120 km/h | Maintain 3–5° glidepath |
| 17 | Decelerate on short final to 80–100 km/h | |
| 18 | Maintain descent rate at 2–3 m/s | Collective controls rate; cyclic controls airspeed |
| 19 | At 3–5 m AGL — begin cushion | Slight collective increase to reduce descent rate |
| 20 | **Main gear contact** at 50–70 km/h | Slight nose-up attitude; mains touch first |
| 21 | **Lower nose** — nosewheel contacts | Forward cyclic as aircraft decelerates |
| 22 | **Collective DOWN** — weight on all gear | |
| 23 | **Toe brakes** — progressive braking | Controlled rollout to taxi speed |
| 24 | **Nosewheel steering** — pedals | Maintain centerline |
| 25 | Clear the runway; taxi to parking or repeat for next pattern | |

### Exercise 2 — Multiple Patterns

Fly 3 additional traffic patterns, each with a running landing. The student must demonstrate consistent performance across multiple approaches.

| Pattern # | Focus Area | Debrief Point |
|---|---|---|
| 2 | Normal pattern, no instructor inputs | Overall smoothness |
| 3 | Intentional slight overshoot — student adjusts or go-arounds | Decision-making |
| 4 | Student-directed — full pattern with self-brief and debrief | Autonomy |

### Exercise 3 — Go-Around

**Setup:** On final approach, stabilized. Instructor announces "go around" at 100 m AGL, or student self-identifies an unstable approach.

| Step | Action | Notes |
|---|---|---|
| 1 | **COLLECTIVE UP** — apply climb power | Arrest descent; torque 80–90% |
| 2 | **CYCLIC FORWARD** — accelerate | Build airspeed for safe flight |
| 3 | Positive rate of climb established | |
| 4 | At 120+ km/h — climbing safely | |
| 5 | **Gear** — consider retraction if long go-around | |
| 6 | Re-enter traffic pattern on upwind | |

**Go-around decision:** If the approach is not stable (airspeed, altitude, descent rate not converging to acceptable) by 100 m AGL — go around. If below 50 m AGL — commit to the landing.

### Exercise 4 — Hover Landing

**Prerequisites:** Hover power check torque ≤ 80%; sufficient power margin confirmed.

**Setup:** Fly a traffic pattern, decelerate to hover on final approach instead of running landing.

| Step | Action | Notes |
|---|---|---|
| 1 | Normal pattern to final approach | 120–140 km/h |
| 2 | Decelerate progressively: 120 → 80 → 50 → 30 km/h | Through ETL — collective UP as speed drops |
| 3 | At ETL (~50 km/h) — increase collective significantly | Power demand rises steeply |
| 4 | **Establish hover** at 3–5 m AGL over intended landing spot | Note torque |
| 5 | If torque > 85% — **abort to running landing** | Accelerate, gain airspeed, run it on |
| 6 | If torque ≤ 85% — lower collective slowly | Descend at < 0.5 m/s |
| 7 | Ground contact — collective FULL DOWN | Weight on gear |
| 8 | Cyclic and pedals neutral — settled | |

### Exercise 5 — Steep Approach

**Setup:** Identify an approach path requiring 8–12° descent angle (simulated obstacle clearance).

| Step | Action | Notes |
|---|---|---|
| 1 | Approach at 100–120 km/h | Slower than normal approach |
| 2 | Increase descent angle to 8–12° | Reduce collective more than normal |
| 3 | Monitor airspeed — maintain ≥ 80 km/h | Do not let speed decay below 80 km/h |
| 4 | As clearing simulated obstacles — increase collective | Arrest the steep descent |
| 5 | Transition to running or hover landing | Depending on power and conditions |

**Caution:** A steep approach at low speed demands high power to arrest the descent. Airspeed below 50 km/h + high descent rate + high collective = settling with power. Maintain airspeed.

### Exercise 6 — Crosswind Pattern

**Setup:** Configure DCS weather for a 6–8 m/s crosswind (within limits; limit is 12 m/s).

| Phase | Crosswind Technique |
|---|---|
| **Downwind** | Crab into wind to maintain parallel track |
| **Base** | Continue crab; track over ground stays on path |
| **Final (long)** | Crab technique; nose not aligned with runway |
| **Short final** | Transition: wing-low technique; into-wind aileron (cyclic) + opposite pedal |
| **Touchdown** | Aligned with runway; main gear touches first; progressive into-wind correction |
| **Rollout** | Nosewheel steering; slight differential braking if needed |

### Shutdown Procedures

After the final landing of Event 7, perform the complete shutdown from Event 2. This is a graded element of Event 7 — the student must demonstrate shutdown without reference to notes.

#### Shutdown Sequence (From Memory)

| Phase | Items |
|---|---|
| **After landing** | Park, parking brake SET, avionics OFF in order |
| **Engine cool-down** | 2 minutes at idle — watch the clock |
| **Engines** | Right engine OFF → wait → Left engine OFF |
| **Rotor** | Nr decays → below 50% → rotor brake applied gradually |
| **WSO cockpit** | Switch to `1` — all weapons SAFE, WSO systems OFF |
| **Pilot cockpit** | Switch to `2` — generators OFF, batteries OFF |
| **Verify** | Cold and dark — both cockpits |

## Completion Standards

| Standard | Requirement |
|---|---|
| **Pattern altitude** | 200 m ±25 m on downwind |
| **Pattern speed (downwind)** | 200–220 km/h |
| **Gear extension** | Completed abeam the runway threshold; 3 green lights confirmed verbally |
| **Final approach speed** | 120–140 km/h on long final; decelerating to 80–100 km/h by short final |
| **Running landing** | Main gear touches first; no skipping; controlled rollout; centerline ±5 m |
| **Go-around** | Positive climb within 5 seconds; no secondary descent; re-patterns correctly |
| **Hover landing (if attempted)** | Within ±2 m of intended point; no bouncing; torque within limits |
| **Steep approach** | Maintains ≥ 80 km/h airspeed; descent rate arrested before runway threshold |
| **Crosswind** | Demonstrates crab + wing-low technique; aligned at touchdown |
| **Shutdown** | Correct sequence from memory; no items omitted; rotor brake after Nr < 50% |

## Common Errors

| Error | Consequence | Correction |
|---|---|---|
| Attempting hover landing without power check | Insufficient power; settles into ground | "Torque check first. If > 85%, run it on. No exceptions" |
| Not extending gear on downwind | Gear-up landing; aircraft damage | "Gear DOWN abeam the threshold. 3 green lights. Say them out loud" |
| Approaching too fast on short final | Overshooting landing zone; excessive float | "120–140 on long final. Decelerating to 80–100 by short final. Not 200 km/h" |
| Approaching too slow on short final | Settling with power; low energy state | "Below 80 km/h early on final = dangerous. Airspeed is energy. Keep it" |
| Not going around when approach is unstable | Dangerous committed landing | "50 m is the wire. Above it — go around. Below it — commit. Nothing in between" |
| Slamming brakes after touchdown | Wheel lockup; directional loss; tire damage | "Progressive braking. Squeeze then hold. Not stomp" |
| Not disconnecting autopilot before pattern | AP fights landing inputs during flare | "AP off on downwind. Hand-fly the approach and landing" |
| Skipping engine cool-down | Turbine thermal shock | "2 minutes. Every time. Engines first, then the debrief" |
