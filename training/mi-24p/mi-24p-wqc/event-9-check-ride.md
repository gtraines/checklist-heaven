## Overview

The Integrated Check Ride is the culminating evaluation of the Mi-24P Weapons Qualification Course. The student demonstrates mastery of all weapons systems — cannon, ATGM, rockets, defensive systems, and air-to-air missiles — in a single, uninterrupted mission scenario.

The check ride is not a collection of individual skills tests. It evaluates whether the student can integrate weapons employment, threat reaction, mission planning, and crew coordination into a coherent combat mission.

| Element | Details |
|---|---|
| **Duration** | 2–3 hours (brief + flight + debrief) |
| **Format** | Oral exam → cockpit inspection → flight evaluation |
| **Grading** | PASS / REPEAT / DOWN per course standards |
| **Prerequisites** | All Events 1–8 complete with PASS grade |
| **Instructor role** | IP occupies back seat (or observes single-player); no hints |

## Pre-Ride Oral Examination

The IP will ask questions from any of the following areas. The student must answer from memory without reference materials.

### Weapons Systems Knowledge

**Required Knowledge:**
- [ ] GSh-30K: caliber, effective range, max ammunition load, limitations
- [ ] Shturm vs. Ataka: range, speed, CEP, warhead differences, when to use each
- [ ] S-5 / S-8 / S-13 / S-24: caliber, effective range, primary warhead, best targets
- [ ] R-60 vs. R-73: seeker type, flare resistance, off-boresight capability, selection criteria
- [ ] SACLOS guidance: how it works; WSO role; platform stability requirements; most common error

### Defensive Systems Knowledge

- [ ] SPO-15: frequency range, what it detects, what it cannot detect
- [ ] MANPADS blind spot: 5 specific threats that produce no RWR warning
- [ ] UV-26: chaff vs. flare selection for 6 different threat types (no errors permitted)
- [ ] Correct CM program for radar SAM response; for IR missile response

### Mission Planning Knowledge

- [ ] Target priority sequence (standard 7-level hierarchy)
- [ ] 3 conditions requiring mission abort
- [ ] Emergency jettison priority sequence
- [ ] Gross weight calculation: given a loadout, state total stores weight, MTOW margin, VNE

### Oral Grading Standards

| Score | Criteria |
|---|---|
| **Satisfactory** | ≥90% of questions answered correctly without hesitation |
| **Repeat** | 80–89% correct, or correct but slow on critical items |
| **DOWN** | <80% correct, or any error on SACLOS principles, CM type selection, or automatic DOWN criteria |

## Ground Phase — Cockpit Inspection

The student performs a complete pre-flight cockpit setup as if preparing for a combat mission.

### Required Sequence

**Weapons Configuration (WSO Seat):**
- [ ] Master Arm — verified SAFE initially
- [ ] ATGM mode selector — correct position for loaded missiles
- [ ] Pylon selectors — verify against loadout
- [ ] Cannon round count — noted

**Defensive Systems (WSO Seat):**
- [ ] SPO-15 — power ON; mode OPERATE; BIT test performed
- [ ] UV-26 — power ON; mode SEMI-AUTO; correct program selected
- [ ] Audio volume — set; functional tone confirmed

**Mission Brief:**
- [ ] Threats briefed (minimum: 1 radar SAM, 1 AAA, MANPADS areas)
- [ ] Targets briefed (minimum: 1 armor, 1 soft target, 1 structure)
- [ ] Route briefed with terrain masking features identified
- [ ] Loadout stated with weight; VNE and hover ceiling stated
- [ ] Bingo fuel defined; abort criteria stated (minimum 2)

## Flight Phase — Integrated Weapons Employment

The IP provides a scenario at the time of the flight — details are not pre-disclosed. All scenarios include elements from every weapon system category.

### Standard Check Ride Scenario (Template)

**Scenario Elements (exact details vary by IP):**

**Target Package:**
- 2–3 MBTs or IFVs (ATGM employment required)
- 2–3 soft vehicles or troop positions (rockets or cannon)
- 1 hardened position or bunker (rockets or ATGM thermobaric)
- 1 ZSU-23-4 or similar radar-guided AAA (must suppress before armor)

**Threat Environment:**
- 1 radar SAM (SA-6 or SA-8 equivalent) — SPO-15 contact during mission
- MANPADS teams at target area — no RWR warning; visual scan required
- Enemy helicopter contact — R-60/R-73 employment opportunity

**Required Demonstrations:**

| Maneuver / Task | Grading Standard | IP Notes |
|---|---|---|
| **Mission brief** | All 8 items complete | Before engine start |
| **Systems check** | SPO-15 + UV-26 online; tested | Pre-departure |
| **NOE ingress** | ≥2 terrain masking features used | 15–50 m AGL |
| **Target identification** | Positive ID via PKV-1 before firing | No random shots |
| **Air defense suppression** | ZSU engaged before armor | Priority sequence |
| **ATGM employment** | Hit or within 5 m on MBT | SACLOS discipline |
| **Cannon employment** | ≥50% rounds on target | Correct profile |
| **Rocket employment** | Correct warhead; correct profile | Weapon match |
| **SPO-15 threat call** | Standard format within 3 sec | SAM contact |
| **CM employment** | Correct type; within 2 sec | Radar or IR |
| **Air-to-air decision** | Correct engage vs. evade call | Threat dependent |
| **BDA report** | Standard format; accurate | Post-attack |
| **Safe egress** | Below 200 m AGL; terrain masking | No loitering |
| **Landing** | Master Arm SAFE before landing | No exceptions |

## Grading Criteria

### PASS

- Oral examination: ≥90% correct
- No automatic DOWN items
- All graded tasks: Satisfactory or better
- No incomplete items (II) carried past second attempt

### REPEAT

- Oral examination: 80–89% correct, OR
- One graded task rated Unsatisfactory (requiring correction), OR
- Any Incomplete Item (II) not corrected on second attempt

A REPEAT requires one additional event re-flying the deficient area. A second REPEAT on the same task is a DOWN.

### DOWN

- Oral examination: <80% correct, OR
- Any Automatic DOWN item triggered (see below), OR
- Two or more Unsatisfactory grades in a single flight, OR
- Inability to correct a REPEAT item

A DOWN requires course review board action and administrative action per local SOP.

### Task Grading Scale

| Grade | Code | Meaning |
|---|---|---|
| **Excellent** | E | Exceeds standard; precise execution |
| **Satisfactory** | S | Meets standard; minor deviations self-corrected |
| **Unsatisfactory** | U | Does not meet standard; IP correction required |
| **Incomplete Item** | II | Could not be completed; opportunity for re-attempt |

---

## Automatic DOWN Criteria

The following errors result in an immediate check ride termination and DOWN grade regardless of all other performance:

| Trigger | Reason |
|---|---|
| **Fratricide (engaging friendly forces)** | Unsafe weapons employment; mission failure |
| **Weapons not SAFE on landing** | Fundamental weapons safety violation |
| **SACLOS seat switch during guidance** | Demonstrated failure to understand guidance principles |
| **Engaging armor before suppressing active AAA threat** | Tactical recklessness; fundamental priority error |
| **Firing outside minimum range (ATGM <400 m)** | Demonstrates lack of system knowledge |
| **Countermeasures type error against active threat** | Fundamental defensive systems failure |
| **Aircraft control loss** | Safety of flight |

## Post-Ride Debrief

### Required Debrief Items

The IP will cover the following in every post-ride debrief regardless of outcome:

1. **Overall performance summary** — Pass / Repeat / Down with rationale
2. **Strongest area** — What the student executed well
3. **Primary deficiency** (if any) — Specific task/standard that was not met
4. **SACLOS performance** — Review of guidance quality data if available
5. **Defensive systems** — CM selection accuracy; RWR response timing
6. **Tactical decisions** — Target priority; weapon selection; abort judgment
7. **Next steps** — Repeat event / graduation / additional training as applicable

### Student Self-Assessment Questions

Before the IP delivers grades, the student self-assesses:
- What was the best-executed maneuver/task in the flight?
- What was the task farthest from standard?
- If you could fly one pass over, which target/weapon would it be and why?

This self-assessment is not graded, but demonstrates situational awareness and judgment — qualities that matter in actual combat.
