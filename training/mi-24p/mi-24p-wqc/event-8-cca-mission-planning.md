## Learning Objectives

Upon completion of this event the student will be able to:
- [ ] Apply Soviet CCA doctrine principles: speed over stealth, mass fire, rapid egress
- [ ] Conduct a mission planning brief covering targets, threats, route, loadout, and abort criteria
- [ ] Execute a low-level nap-of-earth ingress using terrain masking
- [ ] Select and execute the correct attack profile (diving vs. running) based on threat level
- [ ] Sequence multiple targets by priority: air defense → C2 → AT systems → armor
- [ ] Select an appropriate loadout for a given mission type
- [ ] Calculate gross weight with a given loadout and state resulting performance limits
- [ ] Execute emergency jettison by priority sequence
- [ ] Brief formation calls for a two-ship attack sequence (or verbalize in single-player)

## Ground Brief

### The Integrated Challenge

Events 1–7 developed individual weapon systems one at a time. This event integrates everything: picking the right weapon, planning the route, maneuvering to the target, prioritizing what to shoot first, reacting to threats, and getting home. No single weapon proficiency means much if the crew can't put it all together under time pressure.

A complete Mi-24P combat crew must be able to brief a mission, execute an ingress, engage multiple target types in sequence, react to air defenses, and egress — all while managing fuel and ammunition.

## Soviet CCA Doctrine

### Fundamental Philosophy

| Soviet Principle | Implication for Mi-24P |
|---|---|
| **Mass over precision** | Multiple aircraft, high volume of fire; rockets dominate |
| **Speed over stealth** | Fast attack reduces exposure; Mi-24 is not stealthy |
| **Standardized profiles** | Predictable tactics (limitation) but efficient training |
| **Centralized command** | Mission planning done at higher level; crews execute |
| **Combined arms** | Mi-24P always operates with ground forces; not independent |

### Soviet vs. Western Comparison

| Aspect | Soviet (Mi-24P) | Western (AH-64) |
|---|---|---|
| **Approach** | High-speed, low altitude | Standoff; NOE hover |
| **Attack** | Fast diving / running passes | Hover and shoot (ATGM) |
| **Volume** | High (rockets + cannon) | Precision (ATGM primary) |
| **Survivability** | Speed + CM | Stealth + precision |
| **Formation** | Close coordination | More independent |

### Threat Environment Assumptions

Real Soviet Mi-24P doctrine assumed dense, overlapping SAM and AAA. Key assumptions built into every procedure:
- Enemy will detect the aircraft — don't try to hide, try to survive
- MANPADS are everywhere — visual scan at all times
- Air defenses must be suppressed before armor is engaged
- One pass, haul ass — extended loitering over a defended target kills helicopters

## Mission Planning Process

### The 6-Step Planning Brief

**Step 1 — Threat Assessment:**
- Identify known/suspected SAM and AAA positions
- Identify MANPADS threat areas (assume present unless known clear)
- Plan route to avoid highest-threat areas

**Step 2 — Target Analysis:**
- Primary target (mission objective)
- Secondary targets (if primary is destroyed / obscured)
- Opportunity targets (not pre-planned; engage if present and safe)

**Step 3 — Route Planning:**
- Ingress: Use terrain masking; ridgelines, valleys, urban
- Initial Point (IP): Defined position for attack geometry setup
- Egress: Different route than ingress when possible

**Step 4 — Loadout Selection:**
- Match weapons to target types (see loadout table below)
- Balance fuel vs. ordnance for required combat radius
- Confirm stores within weight limits

**Step 5 — Fuel Planning:**
- Mission fuel = leg fuel × 2 (ingress + egress) + combat reserve (15 min minimum)
- Bingo fuel = minimum to return safely; mission abort point

**Step 6 — Contingency Planning:**
- Primary abort criteria: SPO-15 missile guidance on ingress; abort and return
- Secondary abort criteria: System failure that degrades mission-critical capability
- Emergency recovery: Nearest suitable airfield regardless of fuel if aircraft damaged

### Intelligence Requirements

| Type | What to Know | Source |
|---|---|---|
| **SEAD** | SAM/AAA positions and status | Recon, ELINT |
| **Target** | Location, type, priority | Ground FAC, imagery |
| **Weather** | Visibility, wind, ceiling | Met |
| **Friendly Forces** | Positions, planned movements | Ground coordination |

## Tactical Approach Profiles

### Profile 1 — Nap-of-Earth / Low-Level (Primary)

| Parameter | Value |
|---|---|
| **Altitude** | 15–50 m AGL |
| **Airspeed** | 220–260 km/h |
| **Terrain use** | Valleys, behind ridgelines, urban terrain |
| **Formation** | Line abreast or trail; 500–1,000 m separation |
| **Navigation** | Visual pilotage; map/GPS backup |
| **Radio** | Minimal; pre-briefed checkpoints |

### Profile 2 — High-Low Profile

**For long-range approaches over low-threat territory:**
- High phase: 500–1,500 m AGL at cruise speed
- Transition point: Descend 10–15 km from target area
- Low phase: NOE as above for final approach

### Profile 3 — Pop-Up Attack

**For targets masked by terrain:**
- Approach: Low level to planned firing position
- Pop-up: Climb to 200–600 m for line-of-sight
- Attack: Weapon employment from altitude
- Return to low level: Immediate after weapon release

## Target Prioritization

**Standard sequence — engage in this order:**

| Priority | Target Type | Reason |
|---|---|---|
| **1** | Air defense (SAMs, AAA) | Enables all subsequent attacks |
| **2** | Command & Control (command posts, comms) | Disrupts enemy coordination |
| **3** | Anti-tank systems (AT guns, AT missile teams) | Protects friendly armor |
| **4** | Direct fire (tanks, IFVs engaging friendlies) | Immediate threat to ground forces |
| **5** | Indirect fire (artillery, mortars) | Supporting threat |
| **6** | Logistics (supply, fuel, ammunition) | Degrades sustainment |
| **7** | Personnel | Lowest priority; use area weapons |

### Target Identification Sequence

1. Naked eye scan from 2–4 km standoff
2. PKV-1 confirmation — positive ID before engagement
3. Range check — within effective envelope of selected weapon
4. Threat assessment — no immediate air defense threat to the shot
5. Attack authorization — clear with ground controller if CAS

## Attack Profiles

### Diving Attack (Primary — Precision)

| Parameter | Value |
|---|---|
| **Entry Altitude** | 800–1,500 m AGL |
| **Dive Angle** | 20–30° |
| **Entry Airspeed** | 220–260 km/h |
| **Weapon Release Range** | 800–2,000 m (weapon dependent) |
| **Break-off Range** | 400–600 m from target |
| **Recovery Altitude** | 100 m+ AGL minimum |

**Execution:** Approach → identify target → roll in → establish dive → acquire CCIP/pipper → fire at release range → pull out hard → break to egress heading.

### Running Attack (High Threat — Survivability)

| Parameter | Value |
|---|---|
| **Altitude** | 50–200 m AGL |
| **Airspeed** | Maximum safe (260+ km/h) |
| **Track** | Straight line across target area |
| **Shots** | 1 per pass; no re-engagement |
| **Post-pass** | Immediate egress; no BDA on pass |

**When to use:** Dense AAA environment; time-critical target; threat prevents prolonged exposure.

### Multiple Attack Runs

- **First pass:** Highest priority target; most capable weapon
- **Egress:** Brief departure from target area; reposition
- **Second run:** Different axis of attack; secondary target or re-engagement
- **Ammunition:** Reserve sufficient for egress self-defense

## Formation Tactics

### Two-Ship Standard Calls

| Situation | Lead Call | Wingman Response |
|---|---|---|
| **Target sighted** | "Lead has target, tank, 2 o'clock" | "Two has target" |
| **Attacking** | "Lead in hot" | "Two covering" |
| **Weapons away** | "Lead off target, left" | "Two in hot" |
| **Threat warning** | "Flak, 10 o'clock" | "Two breaking left" |
| **Mission abort** | "Lead aborting, fuel" | "Two following" |

### Lead vs. Wingman Responsibilities

| Role | Navigation | Target ID | First Attack | Threat Watch |
|---|---|---|---|---|
| **Lead** | Primary | Primary | Yes | Secondary |
| **Wingman** | Secondary | Secondary | Follow-up | Primary |

## Weight & Performance

### Basic Weight Categories

| Component | Weight (kg) |
|---|---|
| **Empty Weight** | 8,500 |
| **Crew (2×)** | 180 |
| **Fuel (maximum)** | 1,851 |
| **Weapons (maximum)** | 2,400 |
| **MTOW (normal)** | 12,000 |

### Performance vs. Gross Weight

| Gross Weight | Hover OGE Ceiling | VNE (km/h) | Combat Radius (km) |
|---|---|---|---|
| **9,000 kg** | 2,400 m | 335 | 450 |
| **10,000 kg** | 1,800 m | 310 | 420 |
| **11,000 kg** | 1,200 m | 280 | 380 |
| **12,000 kg** | 600 m | 250 | 320 |

**Rule of thumb:** Every 1,000 kg above 9,000 kg costs approximately 600 m of hover ceiling and 20–25 km/h of VNE.

### Asymmetric Loading Warning

Loading one side significantly heavier than the other creates a roll tendency. Limit asymmetric imbalance to ≤100 kg between left and right pylons when possible. If asymmetric loading is unavoidable, brief the pilot before flight.

> *⚠ Realism Divergence: DCS weight/balance is handled automatically by the flight model. Real pilots must verify CG using weight worksheets. DCS may also be more forgiving at maximum weight than the real aircraft.*

## Emergency Jettison

### Jettison Priority Sequence

**Priority 1 — Emergency Performance Recovery:**
- S-24 rockets (235 kg each) — heaviest single items
- B-13L pods + S-13 rockets (435 kg each)

**Priority 2 — Combat Damage:**
- Damaged or malfunctioning stores
- Stores creating asymmetric loading

**Priority 3 — Mission Abort:**
- All unused ordnance per ROE
- Retain minimum: cannon ammunition + 2 AAMs for self-defense egress

### Emergency Jettison Procedure

1. Master Arm → **SAFE** (prevent inadvertent firing)
2. Emergency jettison cover → **LIFT**
3. Jettison button → **PRESS AND HOLD (3+ seconds)**
4. Visual confirmation — all stores separated
5. Trim for new CG position
6. Recalculate performance with new weight

## Post-Attack BDA and Egress

### BDA Standards

| Assessment Type | Method | Timing |
|---|---|---|
| **Immediate** | PKV-1 visual during break | During maneuver |
| **Extended** | PKV-1 from safe standoff | 2–5 km after break |
| **Report** | Standard BDA call to controller | After extended BDA |

**BDA Report Format:** "[Target type] — [Destroyed / Damaged / Missed] — [number destroyed if multiple] — [observed effect]"

### Re-Engagement Decision

| Condition | Decision |
|---|---|
| Target destroyed, no remaining threat | RTB or next target |
| Target damaged, re-engagement possible | Reposition and re-engage |
| Target intact, threat environment high | Abort; report to controller |
| Ammunition critical | RTB; do not waste remaining rounds |

### Egress Planning

- Use different route than ingress when tactically feasible
- Maintain NOE until clear of threat area
- Maximize airspeed initially; cruise for fuel conservation
- Maintain mutual support in formation throughout egress

## In-Sim Procedures

### Mission Brief Checklist (Student Deliverable)

Before each flight in this event, the student must brief the following items to the instructor:

- [ ] Threat assessment — known SAMs, AAA, MANPADS areas
- [ ] Primary and secondary targets — type, location, priority
- [ ] Ingress route — terrain features used; IP defined
- [ ] Loadout — weapons selected; reason for each
- [ ] Gross weight — calculated; VNE and hover ceiling stated
- [ ] Fuel plan — bingo fuel defined; mission abort fuel identified
- [ ] Abort criteria — at least 2 defined scenarios
- [ ] Formation plan (if applicable)

### Mission Practice Scenario — Integrated Attack

**Scenario:** Enemy armored column (4 tanks, 4 BMPs) advancing on friendly position. Position defended by ZSU-23-4 and 2 MANPADS teams. Artillery battery 3 km behind the column.

**Student Tasks:**
1. Select loadout (recommend: 4×Ataka + 2×B-8V20A + 250 rds)
2. Plan ingress route using terrain masking
3. Prioritize targets: ZSU first, then tanks, then BMPs
4. Execute SEAD pass (cannon/rockets against ZSU)
5. ATGM pass against tanks (Shturm or Ataka, distance dependent)
6. S-8 pass against BMPs
7. Egress; report BDA

## Common Errors

| Error | Consequence | Correction |
|---|---|---|
| **Engaging armor before air defense** | Shot down during attack | Always suppress air defense first |
| **Predictable attack axis** | Enemy adapts; increased loss | Vary approach direction pass to pass |
| **Inadequate fuel planning** | Bingo fuel before mission complete | Conservative planning; 15-min reserve |
| **Target fixation** | Miss other threats | Maintain situational awareness throughout |
| **Ammunition waste on low-priority targets** | Cannot engage high-priority targets | Match weapon and quantity to target |
| **Poor egress planning** | Trapped in threat area | Brief egress before attack begins |
| **Single-player seat switching during SACLOS** | Missile miss | ATGM guidance always from WSO seat |
| **No abort criteria** | Continues into unsurvivable situation | Pre-brief and enforce abort criteria |

## Completion Standards

The student must demonstrate the following to the instructor's satisfaction:

- [ ] Complete mission brief covering all 8 items before flight
- [ ] State correct gross weight and VNE for planned loadout
- [ ] Execute NOE ingress using ≥2 terrain masking features
- [ ] Engage air defense (ZSU or equivalent) before engaging armor
- [ ] Execute correct attack profile (diving vs. running) for threat level
- [ ] Engage ≥3 target types using ≥2 different weapon systems correctly matched
- [ ] Provide post-attack BDA report in standard format
- [ ] Complete egress without re-entering target area
- [ ] No fratricide; no weapons safe failure on landing
