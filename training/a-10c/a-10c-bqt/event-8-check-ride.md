## Overview

The check ride is a **full unassisted evaluation flight** flown from cold-and-dark aircraft to full shutdown. The evaluator observes but does **not** assist unless an unsafe act is committed or an unrecoverable safety situation develops.

### Check Ride Profile Summary

| Phase | Events Evaluated |
|---|---|
| Ground | Startup, avionics, EGI alignment, taxi, pre-takeoff checks |
| Departure | Takeoff, clean-up, departure climb |
| Airwork | Level flight, level turns (30° and 45°), climbs/descents, trim |
| Slow flight | Clean configuration; gear + flap configuration |
| Stall | Power-off recognition and recovery (clean, one event) |
| Navigation | CDU steerpoint selection; 2-leg route; HSI course tracking |
| Emergency | Evaluator-called simulated EP (one of: single-engine, hydraulic, or electrical) |
| Pattern | Overhead break approach (1 pattern) |
| Landing | Full-stop landing; rollout; shutdown |

## Pre-Check Briefing

The evaluator briefs the student on the following before the check ride begins:

1. **Purpose:** This is an evaluation of BQT completion standards. It is not a training flight. The evaluator will not coach, correct, or prompt.
2. **Evaluator authority:** The evaluator may terminate the check at any point for an unsafe act, a DOWN item, or pilot incapacitation.
3. **Student authority:** The student may request a clarification of a task before beginning that task only. Once the task is in progress, no assistance will be given.
4. **Emergency:** If a true (non-simulated) emergency occurs, the evaluator will assist. Simulated emergencies are pre-briefed.
5. **Grading:** All items are graded PASS or DOWN. A PASS with noted Incomplete Items (IIs) may be reviewed; a DOWN item requires a REPEAT.
6. **Boldface:** The student will be examined on all 5 boldface procedures verbally before engine start.

---

## Boldface Examination

> This test is conducted verbally, face-to-face, before boarding the aircraft. The student must recite each procedure **word-for-word** from memory. Reference to any written material is **not permitted**.

| Procedure | Pass Criteria |
|---|---|
| **Abort** | All steps in order; no hesitation > 5 seconds |
| **Engine Fire (Inflight)** | All steps in order; no hesitation > 5 seconds |
| **Double Engine Failure** | All steps in order; no hesitation > 5 seconds |
| **Manual Reversion** | All steps in order; no hesitation > 5 seconds |
| **Ejection** | All steps in order; no hesitation > 5 seconds |

> **Failure of any boldface = Check Ride does not proceed.** The student is grounded for additional academic review and re-examination. This is not a DOWN — it is a prerequisite gate.

## Grading Sheets

### Area 1: Ground Operations

| Item | Standard | Result |
|---|---|---|
| Battery → Inverter sequence | Correct order; no steps skipped | PASS / II / DOWN |
| APU start | ITT within limits; stable APU GEN | PASS / II / DOWN |
| Left engine start | No hot start; stable idle; ITT within limits | PASS / II / DOWN |
| Right engine start | No hot start; stable idle; ITT within limits | PASS / II / DOWN |
| Generator transfer | APU GEN off after both gens online | PASS / II / DOWN |
| Avionics sequence | CDU → EGI → UFC → MFCDs in order | PASS / II / DOWN |
| EGI alignment | Stationary during alignment; awaits NAV READY before moving | PASS / II / DOWN |
| Pre-taxi checks | All items complete without prompting | PASS / II / DOWN |
| Taxi | NWS HI on ramp; NWS LO on taxiway; ≤ 15 kts; brake check within 50 ft | PASS / II / DOWN |
| Pre-takeoff checks | Complete; callouts verbalized | PASS / II / DOWN |

**Area 1 DOWN criteria:**
- ITT overtemp (> 860°C) during either engine start without recognition or correction
- Taxiing with NWS HI on taxiways (runway excursion risk)
- Failing to complete EGI alignment before taxi (navigation system unreliable)

### Area 2: Departure & Basic Airwork

| Item | Standard | Result |
|---|---|---|
| Rotation speed | 130–145 KIAS | PASS / II / DOWN |
| Rotation technique | Smooth 2–3°/sec pitch rate; no porpoise | PASS / II / DOWN |
| Gear retraction | On positive rate; all 3 green lights confirmed | PASS / II / DOWN |
| Flap retraction | At ≥ 200 KIAS | PASS / II / DOWN |
| Departure climb | 200–220 KIAS; assigned heading ±10° | PASS / II / DOWN |
| Level flight | Altitude ±150 ft; airspeed ±15 KIAS | PASS / II / DOWN |
| 30° level turn (left and right) | Altitude ±100 ft; ball centered; roll-out within ±15° | PASS / II / DOWN |
| 45° level turn (left and right) | Altitude ±150 ft; ball centered; roll-out within ±15° | PASS / II / DOWN |
| 500 ft/min climb | Rate ±100 ft/min; altitude within 100 ft of target | PASS / II / DOWN |
| 1,000 ft/min descent | Rate ±150 ft/min; altitude within 100 ft of target | PASS / II / DOWN |
| Trim technique | No constant stick pressure required; hands-off verified | PASS / II / DOWN |
| Slow flight (clean) | 140–150 KIAS ±10 KIAS; altitude ±200 ft; 30 seconds | PASS / II / DOWN |
| Stall recognition | Buffet recognized and recovery initiated within 3 seconds | PASS / II / DOWN |
| Stall recovery | Positive altitude trend; < 1,000 ft altitude loss; no secondary stall | PASS / II / DOWN |

**Area 2 DOWN criteria:**
- Gear retraction above 250 KIAS (structural limit exceedance)
- Stall recovery: pulling aft stick after stall break (secondary stall/departure)
- Slow flight below minimum speed (< 120 KIAS clean) without recognition

### Area 3: Navigation

| Item | Standard | Result |
|---|---|---|
| CDU steerpoint selected correctly | Correct STP number; CDI active | PASS / II / DOWN |
| HSI CDI tracking | CDI within half-deflection for ≥ 80% of each leg | PASS / II / DOWN |
| Course intercept technique | Intercept at ≤ 45° cut; no overshoot past CDI center | PASS / II / DOWN |
| TACAN setup | Correct channel; bearing pointer active; Morse ID confirmed | PASS / II / DOWN |
| Steerpoint transition | Next STP selected within 30 seconds of waypoint passage | PASS / II / DOWN |
| TAD situational awareness | Can state position relative to steerpoints when queried | PASS / II / DOWN |

**Area 3 DOWN criteria:**
- Navigation entirely lost (no CDU, no TACAN, and student cannot determine position)
- TACAN tuned wrong station (wrong ID) and used as navigation reference

### Area 4: Emergency Procedures

*One emergency will be introduced by the evaluator inflight. Type is not disclosed to the student pre-flight.*

| Emergency Type | Standard | Result |
|---|---|---|
| **Single-engine** | Correct engine identified within 10 seconds; rudder applied; correct procedure verbalized | PASS / II / DOWN |
| **HYD dual failure** | EFC switch MAN REVERSION selected within 10 seconds; speed reduced; landing planned | PASS / II / DOWN |
| **Double generator** | APU GEN ON within 20 seconds; standby instruments cross-checked | PASS / II / DOWN |

**EP — General Standards:**

| Item | Standard | Result |
|---|---|---|
| Situational awareness | Emergency identified and verbalized correctly | PASS / II / DOWN |
| Boldface steps (if applicable) | Executed in correct order from memory | PASS / II / DOWN |
| Non-boldface steps | Completed using checklist or memory as appropriate | PASS / II / DOWN |
| Aircraft control maintained | Controlled flight throughout; no exceedance of limits | PASS / II / DOWN |
| Recovery plan | Appropriate airfield selected; emergency declared verbally | PASS / II / DOWN |

**Area 4 DOWN criteria:**
- Failure to identify the emergency within 30 seconds
- Executing boldface steps in wrong order
- Loss of controlled flight during EP execution
- Failing to declare an emergency verbally

### Area 5: Traffic Pattern & Landing

| Item | Standard | Result |
|---|---|---|
| Initial speed | 340–360 KIAS at break point | PASS / II / DOWN |
| Initial altitude | 1,400–1,600 ft AGL | PASS / II / DOWN |
| Break bank angle | 55–65° bank; 3+ G applied | PASS / II / DOWN |
| Downwind configuration | Gear + flaps before abeam threshold | PASS / II / DOWN |
| Abeam altitude | 1,400–1,600 ft AGL | PASS / II / DOWN |
| Base turn initiation | Threshold ≈ 45° behind wing | PASS / II / DOWN |
| Final alignment | On centerline within ±15 ft by 500 ft AGL | PASS / II / DOWN |
| Stabilized approach | On-speed and on-glidepath by 500 ft AGL | PASS / II / DOWN |
| AoA indexer — final | Green donut ≥ 20 seconds before threshold | PASS / II / DOWN |
| Touchdown zone | Within first 1,500 ft of runway | PASS / II / DOWN |
| Centerline at touchdown | ±15 ft | PASS / II / DOWN |
| Speed brakes on rollout | Within 3 seconds of mains contact (full-stop only) | PASS / II / DOWN |
| Braking technique | No wheel lockup; progressive; stopped before runway end | PASS / II / DOWN |
| Runway exit | Exited at taxiway; no backtrack required | PASS / II / DOWN |

**Area 5 DOWN criteria:**
- Approach not stabilized by 300 ft AGL (evaluator may call go-around)
- Touchdown beyond first 2,000 ft of runway
- Runway excursion (departing runway surface on landing roll)
- Speed below 120 KIAS on final without immediate go-around

## Unsafe Acts

The following are **immediate termination** criteria regardless of grade:

| Unsafe Act | Definition |
|---|---|
| Airspeed exceedance | > 450 KIAS or structural limit exceeded |
| Altitude violation | Below minimum safe altitude (500 ft AGL) without approach established |
| Landing gear extended above 250 KIAS | Structural limit |
| Flaps extended above 220 KIAS (MVR) or 170 KIAS (DN) | Structural limit |
| Loss of controlled flight | Uncommanded departure from flight |
| Crew incapacitation | Student unable to maintain awareness or control |
| Ground collision risk | Taxi speed or path that would result in collision |
| Ejection criteria met, no ejection | Evaluator may override and call ejection |

> **Unsafe act = Check Ride terminated and recorded as DOWN regardless of performance in all other areas.**

## Pass / Repeat / Down Criteria

| Grade | Criteria |
|---|---|
| **PASS** | No DOWN items; Incomplete Items (IIs) documented but do not prevent PASS |
| **REPEAT** | Any single DOWN item in Areas 1–5; no unsafe acts |
| **DOWN** | Unsafe act committed; two or more DOWN items; any boldface failure |

### Incomplete Items (IIs)

An II is a sub-standard performance that does not meet the standard but is not a DOWN item on its own. IIs are documented and the student is debriefed. Multiple IIs in the same area may be upgraded to a DOWN at evaluator discretion.

| Example II | Notes |
|---|---|
| Flap retraction at 195 KIAS (below 200 minimum) | Minor; not repeated = II |
| Break bank angle of 52° (below 55° standard) | Noted; did not affect pattern geometry = II |
| TACAN Morse ID not confirmed verbally | Procedural; station active = II |
| Speed brakes at 4 seconds post-touchdown | Slightly late; no safety impact = II |

## Post-Check Debrief

The debrief is conducted **immediately** after shutdown. It follows a structured format:

### Debrief Structure

1. **Student self-assessment:** Evaluator asks — *"How do you think the flight went overall?"*
2. **Good items first:** Evaluator notes 2–3 items performed correctly and to standard.
3. **Incomplete Items review:** Each II discussed — what happened, why, and how to correct.
4. **DOWN items (if any):** Discussed factually with specific data (speed, altitude, time) — not opinion.
5. **Unsafe acts (if any):** Reviewed last; recorded in writing.
6. **Grade announcement:** PASS / REPEAT / DOWN with overall summary.
7. **Next step:** If PASS → BQT complete, recommended for mission qualification. If REPEAT → specific events to re-fly identified.

### Grade Documentation

The evaluator completes the following after every check ride:
- [ ] Grading sheet signed by both evaluator and student
- [ ] IIs and DOWN items recorded with event, time, and specifics
- [ ] Unsafe acts (if any) documented with narrative description
- [ ] PASS / REPEAT / DOWN grade entered in student record

## BQT Completion

A PASS on Event 8 constitutes completion of the **A-10C Basic Qualification Training** course. The student is certified as:

> **Qualified — A-10C Basic Aircraft Qualification**
> *Ready to progress to Mission Qualification Training (weapons, threat systems, tactical employment)*
