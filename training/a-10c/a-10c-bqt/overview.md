## Course Description

The A-10C Basic Qualification Training (BQT) course qualifies a pilot to safely operate the Fairchild Republic A-10C "Thunderbolt II" in day VMC conditions from cold-and-dark startup through safe aircraft recovery. Combat employment (weapons delivery, TGP targeting, JTAC coordination) is **not** covered in this course — those topics are addressed in the follow-on A-10C Weapons Employment Course.

> **DCS Module Note:** The DCS A-10C (and A-10C II) is a **full-fidelity, clickable cockpit** module. Unlike the Flaming Cliffs 3 A-10A, every switch, dial, circuit breaker, and panel is interactive. All procedures in this course involve physical cockpit interaction via mouse click and HOTAS command. Keyboard shortcuts exist as alternatives and are noted throughout — but the primary instruction is always **"hands in the cockpit."**

Upon successful completion, the student will be qualified to:
- Perform an independent full cold-and-dark startup including EGI alignment and avionics initialization
- Taxi the aircraft using NWS HI/LO modes, differential braking, and proper speed control
- Execute a normal takeoff, departure climb, and clean configuration by 500 ft AGL
- Conduct basic airwork: coordinated turns, climbs, descents, slow flight, and stall recovery
- Navigate between steerpoints using the CDU, HUD cue, HSI, and basic TAD reference
- Fly a VFR overhead break pattern and execute multiple approaches including go-arounds
- Respond correctly (verbally and procedurally) to all five boldface emergency procedures
- Perform a complete controlled shutdown from engines running to cold-and-dark

## Prerequisites

Before beginning this course, the student must:
- [ ] Have completed the **A-10A BQT** or equivalent fixed-wing qualification, OR demonstrate familiarity with basic fixed-wing aerodynamics (climbs, turns, stalls, pattern work)
- [ ] Understand basic DCS World controls: external view, F10 map, communications menu, pause/unpause
- [ ] Have a functional HOTAS configured: throttle axis, pitch/roll axes, rudder axis or twist
- [ ] Have read the **DCS A-10C or A-10C II flight manual** (Eagle Dynamics in-game or PDF)
- [ ] Be able to use the mouse to click cockpit switches in DCS (hover, left-click, scroll wheel for rotary knobs)

> **Recommended:** Complete the DCS A-10C in-game tutorials (engine start, navigation) before Event 2.

## Syllabus Events

Complete events **in sequence**. Each event must pass before progressing.

| Event | Title | Type | Description |
|---|---|---|---|
| [Event 1](event-1-academics.md) | Aircraft Systems & Academics | Ground | Systems, cockpit layout, limitations, boldface EPs |
| [Event 2](event-2-startup-taxi.md) | Startup & Ground Operations | Flight | Cold-dark startup, EGI alignment, avionics, taxi, pre-takeoff |
| [Event 3](event-3-fam-1.md) | Familiarization Flight 1 | Flight | Takeoff, basic airwork, return and landing |
| [Event 4](event-4-fam-2.md) | Familiarization Flight 2 | Flight | Slow flight, stall recognition & recovery, trim technique |
| [Event 5](event-5-navigation.md) | Basic Navigation | Flight | CDU steerpoints, TACAN, TAD, ILS approach setup |
| [Event 6](event-6-traffic-pattern.md) | Traffic Pattern | Flight | Overhead break, multiple approaches, go-around, crosswind |
| [Event 7](event-7-emergency-procedures.md) | Emergency Procedures | Flight | Simulated single-engine, hydraulic failure, electrical, ejection |
| [Event 8](event-8-check-ride.md) | Check Ride | Evaluation | Full-profile pass/fail evaluation |

**Estimated Completion:** 8–12 simulator sessions.

> **Note:** Events 2 through 8 each begin from cold-and-dark unless otherwise noted. Autostart (`LAlt + Home`) is **not permitted** during graded events — the student must demonstrate the full manual startup. Autostart may be used for self-study only.

## Grading Standards

### Grades

| Grade | Definition |
|---|---|
| **PASS** | All standards met; student may progress to the next event |
| **REPEAT** | One or more standards not met; event must be re-flown before progressing |
| **DOWN** | Unsafe act observed; event grounded pending debrief and instructor review |

### Incomplete Item (II)

An Incomplete Item (II) is issued when the student fails one specific standard but otherwise passes the event. All IIs must be resolved **before the Check Ride (Event 8)**. A student accumulating more than three unresolved IIs is considered not ready for the check ride.

### Grading Tolerances

| Maneuver | Standard |
|---|---|
| Level flight | ±100 ft altitude, ±10 KIAS |
| 30° banked turn | ±100 ft altitude, ball centered |
| 45° banked turn | ±150 ft altitude, ball centered |
| Stall recovery | Recovery within 500 ft altitude loss; no secondary stall |
| Final approach (AoA) | Green donut ±1 indicator for ≥ 15 sec final |
| Touchdown point | Within first 1,000 ft of threshold; on centerline ±15 ft |
| EGI alignment | Full alignment complete before taxi |
| ITT during start | Must not exceed 860°C |
