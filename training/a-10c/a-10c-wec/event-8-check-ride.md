## Purpose and Format

Event 8 is the terminal evaluation for the A-10C Weapons Employment Course. The student demonstrates mastery of all weapon systems trained across Events 1–7 in a single integrated mission. 

| Phase | Format | Duration |
|---|---|---|
| **Academic evaluation** | Oral / written; 10 questions minimum | Pre-flight |
| **Mission planning** | Student plans the profile; examiner reviews | 45–60 min |
| **Airborne evaluation** | Examiner observes; student leads | ~60–90 min flight |
| **Debrief** | Grade determination; standardization review | 30–45 min |

### Course Completion Criteria

To receive a course qualification, the student must:
1. Score ≥ 80% on the academic evaluation.
2. Achieve **PASS** in all airborne task areas.
3. Commit no *UNSAT* or *Automatic Failure* items.
4. Complete all required weapon deliveries within standards.

### Grading Scale

| Grade | Definition |
|---|---|
| **SAT** | Satisfactory — meets or exceeds standards for the task |
| **UNSAT** | Unsatisfactory — does not meet standards; requires remediation |
| **Auto-Fail** | Immediate disqualification (safety violations, weapons safety errors) |

## Academic Evaluation (Pre-Flight)

### Ground Questions — Minimum 10 (Randomly Selected)

Examiner selects at least 10 questions from the following pool. Student must answer ≥ 80% correctly.

#### Systems & Architecture
1. What is the function of the AHCP and name the three arming switches relevant to weapons employment.
2. Describe the DSMS PROF page — what parameters are configured there?
3. What is the difference between CCIP and CCRP delivery modes? When would you use each?
4. What does it mean to "set the SPI" and why is it important before a CCRP delivery?
5. Explain the IFFCC and its role in computing the weapon release point.

#### Guns & Rockets
6. What is the maximum effective range of the GAU-8/A in the A-10C? What dive angle is used in a PAC-1 strafe?
7. What is PAC-2 and when would you use it instead of PAC-1?
8. What are the two Hydra 70 warhead types used in training? What is the purpose of the WP rocket?

#### Bombs
9. You are bombing a lightly armored vehicle with an Mk-82. You have both CCIP and CCRP available. Target is partially obscured. Which mode do you select and why?
10. What is HOF programming on the CBU-97? What happens if you program the wrong HOF?
11. Describe a Snakeye low-altitude pop-up delivery. Why is a pop-up necessary for low-altitude bomb delivery?

#### Guided Weapons
12. How does an LGB home on its target? What happens if the laser is extinguished 5 seconds before impact?
13. What is the laser basket and how does CCRP delivery ensure the LGB is in it?
14. What is the difference between a JDAM in TOO mode and PP mode? Which requires more pre-mission planning?
15. Why is target elevation critical for JDAM employment? What is the consequence of a 200-ft elevation error?
16. Describe the GBU-54 LJDAM. How does it differ from a standard GBU-38?

#### Advanced Systems *(A-10C II students only)*
17. What is APKWS? How does its guidance differ from an unguided Hydra 70 rocket?
18. Describe HMCS target designation. What are two errors a pilot can make when using HMCS?

#### Self-Defense
19. Describe the ALR-69 symbology for a missile launch warning. What is the correct immediate action?
20. What is the difference between chaff and flares? When would you use each?
21. Describe the "notch" maneuver. Against what type of threat is it most effective?
22. What are the three CMSP modes? What mode is recommended for general tactical flight?

## Mission Planning Standards

The student conducts a complete mission brief covering all of the following:

### Required Brief Items

| Topic | Student Must Cover |
|---|---|
| **Threat environment** | Surface threats (radar SAM, IR SAM, AAA) identified; altitudes and defensive reaction plan stated |
| **Loadout** | All stations listed; weapon-to-target matches justified |
| **DSMS profile** | Mode, fuze, quantity for each weapon type |
| **Laser code** | Code stated; CDU LSR programming described |
| **GPS status** | How student will verify GPS READY before first JDAM pass |
| **Attack profiles** | Entry altitude, airspeed, dive angle for each weapon type |
| **Safety rules** | All 5 safety rules recited; minimum release altitudes stated |
| **Recovery plan** | Minimum safe altitude; if weapon fails to release, jettison procedure |

### Mission Sequence Planning (Sample)

The student may propose the following or equivalent:

| Sequence | Target Type | Weapon |
|---|---|---|
| 1 | Stationary vehicle (soft) | GAU-8/A strafe |
| 2 | AFAC marking target | WP Hydra 70 |
| 3 | Armored vehicle cluster | CBU-87 CEM CCRP |
| 4 | Hardened bunker | GBU-12 self-lase |
| 5 | GPS-coordinates fixed target | GBU-38 JDAM TOO |
| 6 | Threat aircraft (defensive) | AIM-9M / defensive maneuver |
| 7 | Self-defense scenario | CMSP, RWR reaction |

## Airborne Evaluation

### Evaluation Conduct Rules

1. **Examiner does not assist** with cockpit procedures — this is a demonstrated proficiency check.
2. Examiner may ask clarifying questions in-flight ("why did you select that fuze?") — student should explain reasoning.
3. Examiner will not intervene unless there is a safety of flight issue.
4. Student is expected to **self-correct minor errors** without prompting — the ability to recognize and correct errors is part of the evaluation.

### Required Airborne Tasks

#### Task Area 1 — Pre-Attack

| Task | Standard |
|---|---|
| Pre-start weapons check | DSMS verified; MASTER ARM SAFE confirmed |
| Post-start power-on sequence | TGP, IFFCC, CMSP powered in correct order |
| CDU setup | Steerpoints loaded; laser code confirmed; GPS verified |
| Pre-IP checks | AHCP set correctly; MASTER ARM ARM at IP only |

#### Task Area 2 — GAU-8/A Strafe

| Task | Standard |
|---|---|
| PAC-1 A or B profile | Correct dive angle and airspeed; roll-in altitude correct |
| Gun rounds on target | ≥ 40% effective hits in 3-second burst |
| Recovery | Min 4,500 ft AGL; 4-G pull; clear of target area |
| SAFE | Gun safe on pull-off |

#### Task Area 3 — Unguided Weapons (Examiner selects one)

| Task | Standard |
|---|---|
| Mk-82 CCIP or CCRP | Impact within 75 m (CCIP) or 75 m (CCRP) |
| CBU-87 CEM HOF programming | Correct HOF entered; DSMS verified before release |
| Snakeye pop-up | Pop initiated at correct altitude; min release altitude observed |

#### Task Area 4 — Precision Guided Munitions — Laser

| Task | Standard |
|---|---|
| GBU-12 CCRP self-lase | Impact within 10 m; laser maintained through impact |
| TGP POINT track confirmed | Before release; stable track demonstrated |
| LASER ARM / MASTER ARM | ARM at IP; SAFE after pass |

#### Task Area 5 — Precision Guided Munitions — GPS

| Task | Standard |
|---|---|
| GBU-38 TOO | GPS verified; TGP POINT track; impact within 30 m |
| GBU-38 PP (if included) | Steerpoint coordinates verified; impact within 30 m |

#### Task Area 6 — Advanced Weapons *(A-10C II Only)*

| Task | Standard |
|---|---|
| APKWS | Impact within 5 m; laser on before impact |
| GBU-54 LJDAM | TGP tracking moving target; laser continuous through impact |

#### Task Area 7 — Self-Defense

| Task | Standard |
|---|---|
| RWR identification | Correctly identifies examiner-described symbol |
| CMSP configuration | Correct mode; chaff or flare selected for threat type |
| Defensive maneuver | Notch within ±15°; break turn ≥ 3 G; flare/chaff dispensed |
| AIM-9M seeker check | Growl tone confirmed; correct acquisition stated before simulated shot |

## Grading Criteria

### Automatic Failure Items

Any of the following result in immediate disqualification regardless of overall performance:

| Item | Reason |
|---|---|
| MASTER ARM ARM before IP without authorization | Safety violation |
| Strafing below 4,500 ft AGL recovery (or < 3,000 ft AGL release) | Safety violation |
| Unintended weapon release (hot trigger) | Safety violation |
| Releasing weapons with unknown SPI (target not designated) | Weapons safety error |
| Laser ARM with MASTER ARM SAFE (laser into aircraft skin) | Safety violation |
| Failure to maintain positive aircraft control at any time | Safety of flight |

### UNSAT Items

Any of the following without successful self-correction result in UNSAT for the task area:

| Item | Definition |
|---|---|
| Wrong fuze selected | Incorrect fuze for target or profile |
| Incorrect DSMS mode | CCIP when CCRP required or vice versa |
| Delivery outside accuracy standard with no self-correction | Impact beyond stated tolerance |
| RWR ignored during threat environment | No reaction within 5 seconds of warning |
| CMSP off during tactical phase | No chaff/flare capability |
| Laser code not verified before LGB pass | Potential dumb bomb delivery |

### SAT Standard

Student achieves SAT for a task area if:
- All required procedures were performed correctly (or minor errors self-corrected).
- Deliveries met accuracy standards per the overview (or student recognized error and requested re-attack).
- No safety items were violated.

## Examiner Notes

### Standard Debrief Format

1. **Safety items first** — any automatic failure items? Debrief these immediately before anything else.
2. **Task area review** — go through each task area in sequence; grade each.
3. **Overall assessment** — did the student demonstrate a consistent, safe, and accurate weapons employment?
4. **Strengths and weaknesses** — provide specific, event-referenced feedback.
5. **Qualification statement** — "I recommend [QUALIFIED / NOT QUALIFIED] for A-10C weapons employment per this course standard."

### Re-Fly Criteria

| Situation | Action |
|---|---|
| Student UNSAT in 1 task area | Remediation flight; re-fly that task area only |
| Student UNSAT in ≥ 2 task areas | Full re-fly of Event 8 after remediation |
| Automatic failure item | Full re-fly; mandatory remediation in specific task area |
| Automatic failure — safety of flight | Grounding review with course director before re-fly |
