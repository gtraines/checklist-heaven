## Course Description

The A-10C Weapons Employment Course (WEC) qualifies a pilot to safely and effectively employ the full range of air-to-ground weapons available on the A-10C and A-10C II Thunderbolt II in DCS World. The course covers every delivery mode and weapon category unique to the A-10C's **full-fidelity avionics suite** — including the DSMS, IFFCC, and Targeting Pod — which are absent from the FC3 A-10A.

> **DCS Module Note:** The DCS A-10C (and A-10C II) is a **full-fidelity, clickable cockpit** module. Unlike the A-10A (FC3), weapons employment requires direct interaction with the DSMS MFD page, AHCP panel, and HOTAS. Every delivery profile, fuze setting, and targeting mode must be configured through the cockpit. Students who attempt to use keyboard shortcuts as their primary interface will find this course significantly more difficult — all instruction uses physical cockpit interaction first.

Upon successful completion, the student will be qualified to:
- Configure the DSMS for any supported weapon type, fuze, delivery mode, and ripple quantity
- Employ the GAU-8/A Avenger in PAC-1 strafe against armored and soft targets
- Deliver unguided rockets (Hydra 70) and gravity bombs (Mk-82, Mk-84) in CCIP diving attacks
- Deliver cluster munitions (CBU-87, CBU-97, WCMD CBU-103/105) with correct HOF and delivery parameters
- Operate the Targeting Pod (Litening AT or Sniper XR) in POINT and AREA track modes, including laser designation
- Employ AGM-65 Maverick (IR and EO-TV variants) using TGP SPI cueing
- Deliver GBU-12/10 Laser-Guided Bombs via self-lase (TGP) and coordinated buddy-lase
- Deliver GBU-38/31 JDAM in TOO and PP modes with correct coordinate accuracy techniques
- **(A-10C II)** Employ APKWS laser-guided rockets, GBU-39 SDB, GBU-54 LJDAM, and HMCS cueing
- Employ AIM-9M Sidewinder for self-defense using the correct tactical decision framework
- Operate the CMSP, ALE-47, and ALQ ECM pod for layered defense in a high-threat environment

> **A-10C II Note:** Events 2–6 apply to both the A-10C and A-10C II. Event 7 (Advanced Systems & Self-Defense) covers systems **exclusive to the A-10C II** (APKWS, GBU-39, GBU-54, HMCS). Students flying the base A-10C should still complete Event 7's self-defense and ECM portions, noting that the A-10C II-specific weapons are marked accordingly.

## Prerequisites

Before beginning this course, the student must:
- [ ] Have **completed the A-10C Basic Qualification Training (BQT)** ([link](../a-10c-bqt/overview.md))
- [ ] Be able to perform an independent cold-and-dark startup through to taxi-ready without reference to checklists
- [ ] Understand DSMS basics: navigating to the DSMS STATUS page and reading station inventory
- [ ] Have read the DCS A-10C Weapons Employment section of the module manual (PDF or in-game)
- [ ] Have reviewed the [A-10C Checklist](../../../checklists/aircraft/a-10c.md) weapons sections
- [ ] Be comfortable using the mouse to click cockpit switches at normal game speed (not paused)

> **Recommended:** Complete the DCS A-10C in-game weapons tutorials (gun/rockets, bombs, Maverick) before Event 2. These are accessible from the DCS Main Menu → Training.

## Syllabus Events

Complete events **in sequence**. Each event must be passed before progressing.

| Event | Title | Type | Weapons / Systems Covered |
|---|---|---|---|
| [Event 1](event-1-academics.md) | Weapons Systems Academics | Ground | DSMS, IFFCC, AHCP, HUD symbology, weapons overview, safety |
| [Event 2](event-2-guns-rockets.md) | GAU-8/A Gun & Rockets | Flight | GAU-8/A (PAC-1), Hydra 70 (M151, M274) |
| [Event 3](event-3-bombs.md) | Unguided & Cluster Munitions | Flight | Mk-82/84 CCIP/CCRP, CBU-87, CBU-97, WCMD CBU-103/105 |
| [Event 4](event-4-tgp-maverick.md) | TGP Operations & AGM-65 Maverick | Flight | Litening/Sniper TGP, AGM-65D/G/H/K/L Maverick |
| [Event 5](event-5-lgb.md) | Laser-Guided Bombs | Flight | GBU-12 (Paveway II), GBU-10 (Paveway II) — self-lase & buddy-lase |
| [Event 6](event-6-jdam.md) | JDAM & GPS-Guided Weapons | Flight | GBU-38 (500 lb JDAM), GBU-31 (2,000 lb JDAM) |
| [Event 7](event-7-advanced-selfdefense.md) | Advanced Systems & Self-Defense | Flight | APKWS, GBU-39 SDB, GBU-54 LJDAM, HMCS (A-10C II); AIM-9M, CMSP, ECM (all) |
| [Event 8](event-8-check-ride.md) | Check Ride | Evaluation | All weapon systems — pass/fail evaluation |

**Estimated Completion:** 10–15 simulator sessions.

> Each flight event begins from **engine-running, taxiing to the arming area** unless otherwise noted. The startup requirement from BQT is assumed to be competent — the WEC does not retest startup procedures. However, students who demonstrate degraded startup proficiency will be required to demonstrate it again before proceeding.

## Grading Standards

| Grade | Definition |
|---|---|
| **PASS** | All standards met; student may progress to next event |
| **REPEAT** | One or more standards not met; event re-flown before progressing |
| **DOWN** | Unsafe act — fratricide, CFIT, weapons release outside safe parameters, or Master Arm failure to safe |

### Incomplete Item (II)
An II is issued when the student fails one weapons category but passes others in the same event. The student must re-demonstrate the failed weapon type before the Check Ride. More than **three unresolved IIs** = not ready for Check Ride.

### Weapons Delivery Standards

| Category | Standard |
|---|---|
| **Gun (PAC-1 strafe)** | At least one confirmed hit per strafe pass; minimum recovery altitude maintained |
| **Rockets (CCIP)** | Impact within 50 m of target; minimum pull-off altitude maintained |
| **Unguided Bombs (CCIP)** | Impact within 75 m of target; correct fuze and arming delay set |
| **Unguided Bombs (CCRP)** | Impact within 50 m of SPI; azimuth line tracking correct before release |
| **Cluster Munitions** | HOF/delivery altitude within spec; correct target type selection (no fratricide) |
| **Maverick (all variants)** | Stable seeker lock confirmed before firing; impact on designated target |
| **LGB (self-lase)** | Bomb impacts within 10 m of TGP designation; laser timing correct |
| **LGB (buddy-lase)** | Correct laser code coordination; "Spot" call made before pickle |
| **JDAM** | Impact within 30 m of designated coordinates; target elevation set via TGP |
| **APKWS** *(A-10C II)* | Impact within 5 m of TGP designation; laser active before firing |
| **SDB / LJDAM** *(A-10C II)* | Weapon impacts designated target within published CEP; laser timing correct for LJDAM |
| **AIM-9M** | Correct tactical decision (engage vs. evade); stable tone before firing |
| **CMSP/ECM** | Correct program selected for threat type; mode appropriate to scenario |

### Safety Rules (Non-Negotiable)
These failures result in an automatic **DOWN** regardless of other performance:

- [ ] Weapons released with Master Arm in SAFE (negligent release)
- [ ] Impact on simulated friendly position (fratricide)
- [ ] Recovery below minimum pull-off altitude on any pass
- [ ] Failure to safe Master Arm before returning to base
- [ ] Release of live ordnance in the arming/de-arming area
