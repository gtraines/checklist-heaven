## Course Description

The A-10A Basic Qualification Training (BQT) course qualifies a pilot to safely operate the Fairchild Republic A-10A "Thunderbolt II" in day VMC conditions from cold-and-dark startup through safe aircraft recovery. Combat employment (CAS tactics, weapons delivery) is **not** covered in this course.

> **DCS Module Note:** The DCS A-10A is a **Flaming Cliffs 3 (FC3)** module with a simplified systems model — there is no clickable cockpit. All systems are operated via keyboard bindings. However, the aircraft uses the **Advanced Flight Model (AFM)**, so its aerodynamic behavior, stall characteristics, and flight envelope are realistic.

Upon successful completion, the student will be qualified to:
- Perform independent cold-start and ground operations
- Taxi using nose wheel steering and differential braking
- Execute a normal takeoff and climb to a working altitude
- Conduct basic airwork: coordinated turns, climbs, descents, slow flight, and stall recovery
- Fly a rectangular traffic pattern and execute multiple approaches and full-stop landings
- Respond correctly to single-engine failure, hydraulic failure, and fire emergency (simulated)

## Prerequisites

Before beginning this course, the student must:
- [ ] Understand basic DCS World controls (external view, map, communications menu)
- [ ] Be able to spawn and load a mission in DCS World
- [ ] Have a functional throttle axis and pitch/roll controls configured
- [ ] Have read the **A-10A FC3 DCS documentation** (Eagle Dynamics manual or in-game)

## Syllabus Events

Complete events in sequence. Each event must be passed before progressing.

| Event | Title | Type | Description |
|---|---|---|---|
| [Event 1](event-1-academics.md) | Aircraft Systems & Academics | Ground | Systems knowledge, limitations, emergency procedures |
| [Event 2](event-2-fam-1.md) | Familiarization Flight 1 | Flight | Cold start, taxi, takeoff, basic airwork, landing |
| [Event 3](event-3-fam-2.md) | Familiarization Flight 2 | Flight | Slow flight, stall recognition & recovery, trim technique |
| [Event 4](event-4-traffic-pattern.md) | Traffic Pattern | Flight | Multiple approaches and landings, go-around, braking |
| [Event 5](event-5-emergency-procedures.md) | Emergency Procedures | Flight | Simulated single-engine, hydraulic failure, ejection criteria |
| [Event 6](event-6-check-ride.md) | Check Ride | Evaluation | Full-profile evaluation flight — pass/fail |

**Estimated Completion:** 5–8 simulator sessions.

## Grading Standards

### Grades
| Grade | Definition |
|---|---|
| **PASS** | All standards met; student may progress |
| **REPEAT** | One or more standards not met; event must be re-flown |
| **DOWN** | Unsafe act observed; event grounded pending debrief |

### Incomplete Item (II)
An II is issued when a student fails to meet a specific standard but passes the overall event. All IIs must be resolved before the Check Ride (Event 6).
