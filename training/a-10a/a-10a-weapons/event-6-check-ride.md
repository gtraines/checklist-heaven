## Check Ride Overview

The Weapons Employment Check Ride is a **pass/fail evaluation** of the full course syllabus. The student flies a pre-planned strike mission against a mixed target set, independently selecting the correct weapon for each target and executing correct employment procedures.

The evaluator provides **no coaching**. The student demonstrates all techniques from Events 1–5 without prompting.

---

## Prerequisites

- [ ] Events 1–5 completed with no unresolved Incomplete Items (IIs)
- [ ] Student can correctly answer weapons-selection questions without reference to notes
- [ ] Student has not received a DOWN grade in any prior event

---

## Grading Criteria

| Grade | Definition |
|---|---|
| **PASS** | All phase standards met; student is Weapons Qualified |
| **INCOMPLETE (II)** | One or two standards not met (no unsafe acts); specific weapons re-demo required |
| **FAIL** | Three or more standards not met, or unable to complete the profile |
| **DOWN** | Unsafe act — fratricide, CFIT, uncontrolled flight, or release below min altitude |

### Automatic DOWN Acts
- Release of any weapon below published minimum altitude
- Engagement of a target that is clearly non-hostile (no VID performed)
- Departure from controlled flight during a weapons pass
- CFIT during attack run
- Firing a Maverick without confirming seeker lock on intended target

## Check Ride Profile

The student will fly the following strike mission against a mixed target set without interruption or coaching:

```
Target Set (pre-briefed):
  A. Truck park (×8 trucks, logistics area) — unguided area weapon appropriate
  B. Armor column (×4 T-72 tanks, moving) — smart area weapon or guided appropriate
  C. Hardened command post (concrete building) — precision heavy weapon required
  D. SAM launcher (SA-8 Gecko, active) — precision standoff required
  E. Single IFV (BMP-2, moving) — gun or precision weapon

Loadout (pre-configured):
  - 2× LAU-68 Hydra 70 M151 HE pods
  - 1× CBU-87
  - 1× CBU-97
  - 2× AGM-65D (IR)
  - 1× AGM-65K (TV Heavy)
  - Gun: 1,174 rounds
```

The student is expected to:
1. Independently assign weapons to targets (no instructor guidance)
2. Execute each attack using correct technique and parameters
3. Employ each weapon with correct switchology

### Suggested Weapon Assignment (Evaluator Reference)

| Target | Preferred Weapon | Rationale |
|---|---|---|
| A. Truck park | CBU-87 or Rockets | Soft targets — area suppression |
| B. Armor column | CBU-97 | Massed armor — Skeets seek tank heat |
| C. Command post | AGM-65K | Hardened target — heavy warhead needed |
| D. SAM launcher | AGM-65D | Precision standoff — stay outside engagement range |
| E. Single IFV | GAU-8/A or AGM-65D | Point target — gun strafing or Maverick |

> **Note:** Other valid combinations exist. The evaluator grades weapon suitability, not memorized answers. Using AGM-65D on the truck park (Target A) would be assessed as poor weapon selection (precision weapon on a soft area target).

3. Schedule re-check ride after minimum one training event per failed category
