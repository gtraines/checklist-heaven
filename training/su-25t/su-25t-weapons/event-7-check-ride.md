## Check Ride Overview

The Weapons Employment Check Ride is a **pass/fail evaluation** of the full course syllabus. The student flies a pre-planned strike mission against a mixed target set, independently selecting the correct weapon for each target and executing correct employment procedures.

The evaluator provides **no coaching**. The student demonstrates all techniques from Events 1–6 without prompting.

---

## Prerequisites

- [ ] Events 1–6 completed with no unresolved Incomplete Items (IIs)
- [ ] Student can correctly answer weapons-selection questions without reference to notes
- [ ] Student has not received a DOWN grade in any prior event

---

## Grading Criteria

| Grade | Definition |
|---|---|
| **PASS** | All phase standards met; student is Weapons Qualified |
| **INCOMPLETE (II)** | One or two standards not met (no unsafe acts); specific weapons re-demo required |
| **FAIL** | Three or more standards not met, or unable to complete the profile |
| **DOWN** | Unsafe act — fratricide, CFIT, uncontrolled flight, or release below min altitude |

### Automatic DOWN Acts
- Release of any weapon below published minimum altitude
- Engagement of a target that is clearly non-hostile (no VID performed)
- Departure from controlled flight during a weapons pass
- CFIT during attack run

## Check Ride Profile

The student will fly the following strike mission against a mixed target set without interruption or coaching:

```
Target Set (pre-briefed):
  A. Vehicle cluster (×4 APCs) — unguided area weapons appropriate
  B. MBT (T-80) — requires tandem-HEAT guided weapon
  C. Hardened command post (concrete) — heavy guided weapon
  D. SHORAD radar (Roland) — SEAD required
  E. Long-range SAM radar (SA-6) — SEAD required

Loadout (pre-configured):
  - 2× B-8M1 S-8 rocket pods
  - 2× Vikhr rails
  - 1× Kh-29L
  - 2× Kh-25MPU
  - 1× Kh-58
  - Fantasmagoria pod (centerline)
  - Gun (250 rounds)
```

The student is expected to:
1. Independently assign weapons to targets (no instructor guidance)
2. Execute each attack in the correct sequence (SEAD before strike, if applicable)
3. Employ each weapon with correct switchology and technique

## Post-Check Ride Debrief

### If PASS

The student is **Weapons Qualified on the Su-25T Frogfoot-B**.

Recommended follow-on training:
- [Basic & Intermediate Surface Attack Course](../surface-attack-course.md) — tactical integration of weapons with attack patterns and JTAC coordination
- [NTTR Surface Attack Training Scenarios](../nttr-training-scenarios.md) — practical mission application

### If FAIL or DOWN

1. Debrief each failed item — identify root cause (knowledge gap, technique, or weapons selection)
2. Return to the specific event covering the failed weapon type and re-fly as training
3. Schedule re-check ride after minimum one training event per failed category
