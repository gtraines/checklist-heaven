## Course Description

The UH-1H Huey Basic Qualification Training (BQT) course qualifies a pilot to safely operate the Bell UH-1H Iroquois in day VMC conditions from cold-and-dark startup through safe aircraft recovery. Combat employment (weapons delivery, door gun operations, rocket employment, tactical flight, NOE/contour operations) is **not** covered in this course.

> **DCS Module Note:** The DCS UH-1H (Belsimtek / Eagle Dynamics) is a **full-fidelity, clickable cockpit** module with an Advanced Flight Model (AFM). Every switch, dial, circuit breaker, and instrument is interactive. All procedures in this course involve physical cockpit interaction via mouse click. The module simulates a realistic analog cockpit — there are no MFDs, no GPS, and no digital flight management systems.

The UH-1H cockpit is **fully analog**. This course therefore emphasizes skills that are not required in glass-cockpit aircraft: manual engine monitoring via individual steam gauges, manual radio tuning, radio-aid navigation (ADF, VOR/ILS), and throttle/RPM management using the correlator and governor system.

Upon successful completion, the student will be qualified to:
- Perform an independent full cold-and-dark startup through engine running and rotor engagement without exceeding EGT limits or missing any start abort criterion
- Perform a stationary hover and hover taxi with correct translating tendency compensation
- Execute a normal takeoff including transition through ETL and climb departure
- Conduct basic airwork: coordinated turns, climbs, descents, slow flight, force trim technique
- Navigate between points using ADF, VOR, and dead reckoning; tune and interpret all three radios (VHF-AM, UHF-AM, FM)
- Fly a standard VFR helicopter traffic pattern and execute multiple approach types including go-arounds
- Respond correctly (verbally and procedurally) to all boldface emergency procedures
- Perform a practice autorotation to a safe recovery
- Complete a controlled full shutdown from engine running to cold-and-dark

## Prerequisites

Before beginning this course, the student must:
- [ ] Have completed **basic rotary-wing familiarization** or demonstrate hover proficiency in any DCS helicopter — **this is not a first-helicopter course**
- [ ] Understand basic DCS World controls: external view, F10 map, communications menu, pause/unpause
- [ ] Have a functional HOTAS configured with collective axis, cyclic axes (pitch/roll), and anti-torque pedals or twist-rudder
- [ ] Be able to use the mouse to click cockpit switches in DCS (left-click for toggles, mouse-wheel for rotary knobs)

> **Recommended:** Achieve a stable unassisted 3-foot hover for 30 seconds before attempting Event 2. If this is not achievable, practice hover control in the DCS free flight option before proceeding.

> **Note on Pedals:** Anti-torque control is critical in the UH-1H due to the single main rotor and high engine torque at takeoff power. A twist-rudder joystick is the minimum recommended setup. Dedicated rudder pedals are strongly preferred. Keyboard rudder (`Z`/`X`) is workable but makes precise hover yaw control difficult — expect additional practice time.

> **Note on Throttle:** The UH-1H uses a twist-grip throttle on the collective handle. In DCS this is operated by **mouse wheel on the 3D cockpit twist-grip** or by mapping a physical throttle axis. The collective (`PgUp`/`PgDn`) and throttle are **separate controls** — do not confuse them.

## Syllabus Events

Complete events **in sequence**. Each event must pass before progressing.

| Event | Title | Type | Description |
|---|---|---|---|
| [Event 1](event-1-academics.md) | Aircraft Systems & Academics | Ground | T53 engine, rotor system, all aircraft systems, analog cockpit layout, limitations, boldface emergency procedures |
| [Event 2](event-2-startup-ground-ops.md) | Startup & Ground Operations | Flight | Cold-dark startup, rotor engagement, hover taxi, shutdown |
| [Event 3](event-3-hover.md) | Hover & Hover Taxi | Flight | Stationary hover, pedal turns, translating tendency, hover power check, hover taxi |
| [Event 4](event-4-fam-flight.md) | Familiarization Flight | Flight | Normal takeoff, ETL transition, basic airwork, trim technique, normal approach and landing |
| [Event 5](event-5-navigation.md) | Basic Navigation | Flight | Radio tuning, ADF, VOR/ILS, dead reckoning, cross-country nav profile |
| [Event 6](event-6-traffic-pattern.md) | Traffic Pattern & Landings | Flight | VFR pattern, multiple approach types (normal, steep, running), go-arounds |
| [Event 7](event-7-emergency-procedures.md) | Emergency Procedures | Flight | Engine failure, practice autorotation, governor failure, hydraulic failure, tail rotor failures |
| [Event 8](event-8-check-ride.md) | Check Ride | Evaluation | Full-profile pass/fail evaluation |

**Estimated Completion:** 8–14 simulator sessions.

> **Note:** Events 2 through 8 each begin from cold-and-dark unless otherwise noted. Autostart is **not permitted** during graded events — the student must demonstrate the full manual startup. Autostart may be used for self-study only.

## Grading Standards

### Grades

| Grade | Definition |
|---|---|
| **PASS** | All standards met; student may progress to the next event |
| **REPEAT** | One or more standards not met; event must be re-flown before progressing |
| **DOWN** | Unsafe act observed; event grounded pending debrief and instructor review |

### Incomplete Item (II)

An Incomplete Item (II) is issued when the student fails one specific standard but otherwise passes the event. All IIs must be resolved **before the Check Ride (Event 8)**. A student accumulating more than three unresolved IIs is considered not ready for the check ride.

### Grading Tolerances

| Maneuver | Standard |
|---|---|
| **Stationary hover (< 5 kts wind)** | ±2 ft altitude, ±3 ft lateral/longitudinal drift over 30 sec |
| **Hover altitude (general)** | 3–5 ft skid height maintained |
| **Pedal turn** | ±5° of target heading, ±2 ft altitude, no drift |
| **Level flight** | ±100 ft altitude, ±5 KIAS |
| **Coordinated turn** | Ball centered throughout; ±100 ft altitude |
| **Autorotation entry Nr** | 294–324 RPM maintained at entry; no Nr below 294 RPM |
| **Approach angle (normal)** | 8–12° glidepath; deceleration to hover at designated point |
| **Approach angle (steep)** | 15–20° glidepath maintained |
| **Hover to touchdown** | Smooth set-down; no dynamic rollover tendency; both skids simultaneously |
| **Engine start EGT** | Must not exceed 720°C starting limit; monitor continuously |
