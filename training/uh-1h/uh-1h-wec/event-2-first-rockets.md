## Learning Objectives

Upon completion of this event the student will be able to:

- [ ] Complete the full arming checklist verbally, in order, from memory, while maintaining aircraft control in the target area
- [ ] Adjust sight depression to a specified value using the depression knob without looking away from the cockpit panel for more than 3 seconds
- [ ] Conduct level / running fire at 80 KIAS; observe rocket impact relative to aim point; describe the error (long/short/left/right)
- [ ] Execute a diving attack pass at 10°, 80 KIAS, releasing at 800–1,000 m slant range, with pull-off above 300 ft AGL
- [ ] Observe own rocket impact and state the correction required to the depression setting or aim point
- [ ] Complete the safe checklist immediately after the last round, in correct order, with Master Arm SAFE as the first action

## Ground Brief

### Mission Profile

1. Normal startup from cold-dark (student unaided)
2. Hover check — read torque; state power margin; GO/NO-GO call
3. Armed departure to designated range area
4. Arming sequence — verbal drill at cruise altitude before entering the range
5. IP demonstrates: set depression 35 mils (level fire, 80 KIAS, 800 m); explain the sight
6. Student: 3 passes level fire; observe and verbalize impact; no correction required for grading on this pass
7. IP demonstrates: 10° diving attack — complete run from roll-in through pull-off; talk through parameters
8. Student: 3 diving passes at 10°, 80 KIAS; verbal correction after each; IP scores
9. Safe sequence after each engagement; verbal drill
10. Return to base; normal landing; shutdown

### Key Numbers Table

| Parameter | Value | Notes |
|---|---|---|
| **Attack airspeed — level fire** | 80 KIAS | Stabilize before the range boundary |
| **Depression — level fire, 80 KIAS, 800 m** | 35 mils | Set before entering the range |
| **Attack airspeed — diving (10°)** | 80 KIAS | Stabilize at roll-in |
| **Depression — diving 10°, 80 KIAS, 800 m** | 26 mils | |
| **Depression — diving 10°, 80 KIAS, 1,000 m** | 32 mils | Primary training value |
| **Roll-in altitude** | 800–1,200 ft AGL | Establish orbit at this altitude before roll-in |
| **Minimum release altitude (10° dive)** | 300 ft AGL | Hard floor — pull off at or before this altitude |
| **Pull-off G** | 2–3 G | Coordinated pull; right pedal to counter torque |
| **Orbit airspeed** | 60–80 KIAS | After pull-off; climbing back to orbit altitude |
| **Fire mode** | SINGLE or PAIRS | SINGLE for observation passes; PAIRS for standard training |
| **Rocket load (standard this event)** | 2× XM-158 (14 total) | 7 per pylon; conserve for multiple passes |

### Weapons Safety Review (Verbalize Before Every Pass)

Before every attack run, regardless of how many times it has been done before, the student verbalizes:
> *"Weapon selector [position], mode [setting], guard lifted, Master Arm ARM, panel clear, sight set [X] mils."*

Before every trigger press:
> *"Weapons hot, engaging."*

After every last round:
> *"Trigger released, Master Arm SAFE, selector SAFE, crew — weapons safe."*

The IP will grade every arming and safe sequence on every pass. A missed step or out-of-order call is an Incomplete Item (II). A weapons fire without arming verbalization is a DOWN.

## Part 1 — Startup & Armed Departure

### Pre-Flight

1. DCS Mission Editor loadout: **2× XM-158, M151 HE rockets; 2× M60D door guns**.
2. Student performs full cold-dark startup without IP assistance (reference BQT Event 2 procedure).
3. After engine running and rotor engaged: pre-departure checks including hover power check.

### Hover Power Check (With Weapons Loaded)

- [ ] Hover IGE at 3–5 ft skid height.
- [ ] Read torque gauge: state the value.
- [ ] State power margin: "Torque is [X] PSI; limit is 40 PSI; margin is [Y] PSI."
- [ ] If torque ≤ 35 PSI: *"Power check good — GO for departure."*
- [ ] If torque > 38 PSI: "Performance marginal — consider load reduction."
- [ ] If torque > 40 PSI: abort — reduce fuel or weapons load before departure.

### Departure

- Depart the airfield toward the designated range area.
- Climb to 1,000 ft AGL cruise altitude; 80 KIAS transit airspeed.
- IP briefs the range entry procedure during transit.

## Part 2 — Arming Sequence in the Air

### Range Entry Procedure

At 2–3 nautical miles from the range boundary:

1. **Student:** Reduce airspeed to 80 KIAS; establish level flight at 1,000 ft AGL.
2. **Student:** Execute the full arming checklist verbally:

> 1. *"Weapon selector — ROCKETS."* (click selector to ROCKETS)
> 2. *"Fire mode — SINGLE."* (click intervalometer to SINGLE)
> 3. *"Master Arm guard — lifted."* (left-click guard to raise)
> 4. *"Master Arm — ARM."* (left-click switch to ARM)
> 5. *"Armament panel — scanning — selector ROCKETS, mode SINGLE, all correct."*
> 6. *"Crew — weapons hot, range entry in progress."*
> 7. *"Sight — head on sight — reticle visible."*

3. **IP confirms:** arming checklist complete and correct; clears student to proceed to range.

> **IP note:** On the first two passes this event, the IP should allow one prompted correction if the student misses a step or reverses the order. On passes 3+, any out-of-order or missed step scores as an II.

## Part 3 — Sight Depression Familiarization

Before firing, take 5–10 minutes to familiarize with sight adjustment.

### IP Demonstration

1. IP takes control; orbits the range at 800 ft AGL, 60 KIAS.
2. IP adjusts depression knob using mouse wheel while explaining:
   - *"I'm rolling the knob clockwise — watch the reticle — it's moving down relative to the aircraft nose. Higher depression = sight points further below the nose = fires at closer range."*
   - *"I'm rolling counter-clockwise — depression decreasing — sight points closer to the horizon — fires at longer range."*
3. IP sets depression to **32 mils** (standard training value: 80 KIAS, 1,000 m, 10° dive).
4. IP explains: *"At 80 KIAS on a 10° dive, with this depression set, if I align the center dot on the target and fire when the slant range reaches 1,000 meters, the rocket should impact near the aim point."*

### Student Practice

1. Student takes control.
2. IP calls a depression value: **35 mils**, **26 mils**, **32 mils** — student adjusts the knob and reads the value back.
3. Student must set and read each value within 5 seconds without losing aircraft control.
4. Repeat until student can set any value ± 2 mils on command.

## Part 4 — Level / Running Fire (Familiarization)

### Purpose

Level fire is used to observe rocket trajectory and impact point in a low-pressure setting before introducing diving attack geometry. Accuracy is lower in level fire — that is expected. The goal here is to observe the ballistic arc and practice trigger discipline.

### Profile

- Altitude: 200–300 ft AGL
- Airspeed: 80 KIAS
- Depression: 35 mils (set before beginning)
- Engagement range: 800 m from target
- Fire mode: SINGLE (one rocket per pass for observation)

### Procedure

1. **Identify target** from 2,000 m out; plan approach heading.
2. **Set depression**: 35 mils on the sight knob.
3. **Approach**: descend to 200–300 ft AGL; maintain 80 KIAS.
4. **At ~1,200 m from target**: align nose on target; confirm sight reticle aligned.
5. **At ~800 m slant range**: press trigger once; observe rocket arc.
6. **Continue forward**: do not pull up directly over target; offset 100 m laterally after firing.
7. **Verbalize impact**: *"Impact was [long/short/left/right] by approximately [distance]."*
8. **Safe sequence**: Execute full safe checklist verbally.
9. **Re-arm for next pass**: if re-attacking, execute full arming checklist again.

### Observation Table (Reference for Impact Correction)

| Observation | Probable Cause | Correction |
|---|---|---|
| Impact short of target | Depression too steep OR airspeed too slow | Reduce depression by 3–5 mils; verify 80 KIAS |
| Impact long of target | Depression too shallow OR airspeed too fast | Increase depression by 3–5 mils; verify 80 KIAS |
| Impact left of target | Aim point right of centerline OR right crosswind | Adjust left aim point offset; note crosswind |
| Impact right of target | Aim point left of centerline OR left crosswind | Adjust right aim point offset; note crosswind |

## Part 5 — First Diving Pass (10° Dive)

### IP Demonstration (Full Talk-Through)

Before the student dives:

1. IP takes control; climbs to 1,200 ft AGL orbit at 70 KIAS.
2. IP talks through every phase:
   - *"I'm in the orbit at 1,200 ft, 70 KIAS. The target is at my 12 o'clock. I'm setting depression to 32 mils for 80 KIAS at 1,000 m range."*
   - *"At about the 9 o'clock position in the orbit — 90° before my attack heading — I'll begin rolling in."*
   - *"Rolling in now — I'm lowering the nose to establish 10° dive. Accelerating through 80 KIAS."*
   - *"Sight is tracking to the target — I'll fire when slant range is approximately 1,000 m."*
   - *"Fire! Pulling off — 2 G pull — break right — climbing. You'll see the impact as I break."*
   - *"Impact observed. Now: Master Arm SAFE, selector SAFE — weapons safe."*
3. IP returns control to student.

### Student Diving Pass Procedure

1. **Establish orbit**: 1,000–1,200 ft AGL; 60–70 KIAS; orbit radius ~800–1,000 m from target.
2. **Set depression**: 32 mils (80 KIAS, 1,000 m, 10° dive) — do this during the orbit, not during the dive.
3. **Arming checklist**: execute full verbal checklist in the orbit before roll-in.
4. **Roll-in point**: at 90–120° before the attack heading; begin inbound turn; lower nose to 10°.
5. **Stabilize**: 80 KIAS; 10° dive; align sight on target.
6. **Fire**: single rocket at approximately 1,000 m slant range.
7. **Pull-off**: immediately after firing — 2–3 G cyclic pull; break left or right; climb.
8. **Floor**: 300 ft AGL — **do not press below this altitude under any circumstances**.
9. **Observe impact**: during pull-off, note impact point; verbalize.
10. **Safe sequence**: immediately after the break.

### Student Pass Requirements (This Event)

- 3 diving passes minimum.
- After each pass: student verbalizes impact observation and what correction will be made.
- After the third pass: student must be within 50 m of the aim point on at least one pass.
- Pull-off must be above 300 ft AGL on all passes.

## Part 6 — Observation, Correction & Re-Attack

### The Observe-Correct-Re-Attack Cycle

After each pass:

1. **Observe**: where did the rockets impact relative to the aim point?
   - *"Impact was approximately 100 m short and slightly left."*
2. **Correct**: what change to depression or aim point is needed?
   - *"I'll increase depression by 4 mils and offset aim point slightly right."*
3. **Re-arm**: execute full arming checklist before the next pass.
4. **Re-attack**: fly the corrected pass; observe new impact.
5. **Verbalize** each step aloud — the IP needs to hear the correction rationale.

### Correction Rules

| Error | Correction Action |
|---|---|
| Rockets short | Reduce depression 3–5 mils (or: verify airspeed was 80 KIAS) |
| Rockets long | Increase depression 3–5 mils |
| Rockets left | Offset aim point right; check for right crosswind |
| Rockets right | Offset aim point left; check for left crosswind |
| Airspeed error at fire point | Do not correct depression — fix the airspeed on re-attack and re-set depression to table value |

> **IP note:** Students often want to correct by changing airspeed rather than depression, or by changing depression to compensate for airspeed error. Reinforce the hierarchy: airspeed first (match the table); then depression setting; then aim point offset. Correcting the wrong variable teaches bad habits.

## Part 7 — Safe Sequence & Return

### Post-Engagement Safe Sequence

After the final pass:

1. **Trigger** — RELEASE
2. **Master Arm** — SAFE (immediately; first action)
3. **Master Arm guard** — LOWER
4. **Weapon selector** — SAFE
5. **Crew** — *"Weapons safe — range departure."*
6. **Jettison panel** — confirm cover closed

### Range Departure

- Climb to 1,000 ft AGL transit altitude.
- Advise IP: *"Range departure complete, [X] rockets expended, [Y] remaining, weapons safe, departing south."*
- Return to airfield; normal approach and landing; full shutdown.

### Post-Flight Debrief Points

1. Were all arming and safe sequences complete and in order?
2. What was the observed impact error on each diving pass?
3. What correction was made? Was it the correct variable?
4. Was the pull-off above 300 ft AGL on every pass?
5. Was airspeed stable at 80 KIAS at the fire point?

## Completion Standards

| Standard | Criterion |
|---|---|
| Hover power check | Torque stated; GO/NO-GO verbalized before departure |
| Arming checklist | Complete, in order, verbalized before every pass; no omissions |
| Safe checklist | Master Arm SAFE is first action; complete and in order after every engagement |
| Sight depression | Set to ±2 mils of table value before each pass; verbalized |
| Level fire | At least 2 passes at 80 KIAS; impact verbalized each pass |
| Diving pass — altitude | Pull-off above 300 ft AGL on all 3 passes — **zero tolerance** |
| Diving pass — accuracy | At least 1 of 3 passes impacts within 50 m of aim point |
| Diving pass — airspeed | 70–90 KIAS at fire point on all passes |
| Observation/correction | Student correctly identifies error and states the right correction after each pass |
| Post-engagement safe | Safe sequence executed correctly after final pass |

## Common Errors

| Error | Correction |
|---|---|
| Arming sequence started during the dive (not during orbit) | Enforce: arming checklist complete before roll-in; if not armed at roll-in, do not roll in |
| Depression not changed from level fire (35 mils) before diving pass | Brief: different profile = different depression; set before entering the orbit, not at roll-in |
| Pressing below 300 ft AGL before fire point | Hard altitude callout: "Pulling off at 400 ft" on every pass; IP intervenes at 350 ft |
| Firing at wrong range (too close = late; too far = early) | Brief the slant range indicator or use a landmark/reticle size estimate; practice range estimation |
| Not verbalizing impact after each pass | Make verbalization a graded item; no credit for the pass without an impact call |
| Safe sequence: selector to SAFE before Master Arm | The first action is always Master Arm SAFE — before selector, before radio, before anything |
| Trying to correct impact error by changing airspeed | Reinforce: airspeed follows the table; depression corrects for impact error |
