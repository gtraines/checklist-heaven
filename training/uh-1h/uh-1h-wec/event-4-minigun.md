## Learning Objectives

Upon completion of this event the student will be able to:

- [ ] State the M134 system characteristics: rate of fire, effective range, and ammunition load from memory
- [ ] Select the minigun as the active weapon on the armament panel and execute the arming checklist for minigun
- [ ] Set the correct sight depression value for minigun and describe why minigun depression differs from rockets at the same airspeed and range
- [ ] Conduct a diving attack with minigun; track the target with tracer walk; correct fire to impact
- [ ] Conduct running fire with minigun; use tracer trace to walk rounds onto target
- [ ] Apply burst control; state the maximum burst duration and explain the consequence of sustained fire

## Ground Brief

### Mission Profile

1. M134 knowledge review (verbal quiz, 3 questions)
2. Cold-dark startup; arm for minigun before departure
3. Range transit at 1,000 ft AGL, 80 KIAS
4. IP demo: minigun diving attack — full talk-through
5. Student: 3 diving attacks; tracer walk technique; verbal impact observation each pass
6. Running fire: 2 passes; tracer walk to target
7. Ammo management drill: IP calls remaining rounds at mid-event; student plans remaining passes
8. Safe sequence; return; debrief

### Key Numbers Table

| Parameter | Value | Notes |
|---|---|---|
| **M134 rate of fire** | ~2,000–4,000 rpm (DCS: ~3,000 rpm) | Variable in real weapon; modeled as high-rate |
| **Effective range** | 800–1,000 m | Beyond 1,000 m, dispersion degrades accuracy significantly |
| **Ammunition per gun** | 3,000 rounds (DCS loadout typical) | 6,000 rounds total for twin minigun loadout |
| **Tracer ratio** | 1:5 (1 tracer per 5 rounds) | In DCS, visible tracer stream |
| **Depression — 80 KIAS, 800 m, 10° dive** | 26 mils | Same table as rockets at equivalent speed/range — depression values are shared |
| **Depression — 80 KIAS, level fire** | 35 mils | |
| **Minimum range** | 200 m | Danger close distance for 7.62mm |
| **Max burst duration (real weapon)** | 4–5 seconds | Barrel temperature management |
| **DCS burst discipline** | No barrel overheat modeled | Still practice 3–5 s burst discipline as a procedure |
| **Min pull-off alt (10° dive)** | 300 ft AGL | Same as rockets |
| **Min pull-off alt (20° dive)** | 500 ft AGL | Same as rockets |

### Minigun vs. Rockets — Key Differences

| Factor | Rockets (FFAR) | M134 Minigun |
|---|---|---|
| Effect on target | High-explosive point detonation | 7.62mm ball/tracer; penetration only |
| Best use | Hardened targets, vehicle kills, area fire | Personnel, soft targets, suppression |
| Trajectory | Ballistic arc — set depression before firing | Similar ballistic rules; tracer walk allows real-time correction |
| Correction method | Observe impact; adjust depression | Walk tracers onto target while firing |
| Ammunition expenditure | High (rockets are discrete) | Very high (3,000 rounds exhausted in ~60 s of continuous fire) |

## Part 1 — M134 System Knowledge Review

### Verbal Quiz (3 Questions — Required Before Departure)

IP asks the following; student must answer all three correctly:

1. **"What is the M134 effective range?"**
   - Answer: 800–1,000 m. Beyond 1,000 m, dispersion degrades accuracy significantly.

2. **"What is the danger close distance for M134?"**
   - Answer: 200 m from friendly forces.

3. **"How does minigun burst duration affect the weapon in the real aircraft?"**
   - Answer: Sustained fire heats the barrels; real-world limit is approximately 4–5 seconds per burst to prevent barrel overheating. In DCS, overheating is not modeled, but burst discipline is practiced as a procedural standard.

### Additional Knowledge Points

**Why use minigun over rockets?**
- Personnel in the open
- Soft-skin vehicles
- Suppression of crew-served weapons
- Target behind cover where rocket detonation is less effective

**Why not minigun in all cases?**
- No explosive effect — no fragmentation
- Poor effectiveness against hardened or armored targets
- Rapid ammunition expenditure limits engagement duration

## Part 2 — Arming & Safe Sequence for Minigun

### Armament Panel Setup — Minigun

1. **Weapon selector** — GUNS (or GUN, depending on panel labeling in DCS)
2. **Fire mode / trigger** — standard trigger (trigger axis or `Space`)
3. **Master Arm guard** — LIFT
4. **Master Arm** — ARM
5. **Panel check** — selector on GUNS; mode correct; panel clear
6. **Crew call** — *"Weapons hot — minigun selected — range entry."*

> **DCS Note:** In the DCS UH-1H, the weapon selector on the armament panel switches between ROCKETS and GUNS. Confirm the selector position visually before arming. If both rockets and minigun are loaded, you must select the correct system. Firing rockets when minigun is intended (or vice versa) is an error even if it does not cause a safety incident in DCS — treat it as a real-world procedural failure.

### Arming Checklist — Minigun (Verbal)

> 1. *"Weapon selector — GUNS."*
> 2. *"Fire mode — continuous trigger."*
> 3. *"Master Arm guard — lifted."*
> 4. *"Master Arm — ARM."*
> 5. *"Panel scan — selector GUNS, correct."*
> 6. *"Crew — weapons hot, minigun, range in."*

### Safe Sequence — Minigun (Verbal)

> 1. *"Trigger — RELEASED."*
> 2. *"Master Arm — SAFE."* (first action — always)
> 3. *"Master Arm guard — LOWER."*
> 4. *"Weapon selector — SAFE."*
> 5. *"Crew — weapons safe."*

## Part 3 — Depression Table & Tracer Walk

### Depression Values (Same Table as Rockets)

The M134's ballistic trajectory at identical speed and range values follows the same depression table as the 2.75" FFAR rockets. Set depression before the pass using the same three-table system.

| Profile | Depression |
|---|---|
| Level fire, 80 KIAS, 800 m | 35 mils |
| 10° dive, 80 KIAS, 800 m | 26 mils |
| 10° dive, 80 KIAS, 1,000 m | 32 mils |
| 20° dive, 80 KIAS, 1,000 m | 22 mils |

### Tracer Walk

Unlike rockets (where you fire and observe the impact), the minigun allows real-time correction during a burst:

1. **Pre-set depression** based on table.
2. **Begin burst** — observe tracer stream.
3. **If tracers are short of target**: lower the nose slightly to walk rounds forward.
4. **If tracers are long (over target)**: raise the nose slightly to walk rounds back.
5. **When tracers are on target**: hold — maintain aim until burst complete or pull-off.
6. **Do not over-correct**: small inputs (1–2°) are sufficient.

> **IP Note:** Tracer walk is the key difference in technique between rockets and minigun. Rockets require pre-run setup and a single fire event; the minigun is interactive. Students must learn to trust the tracer trace rather than the pre-set depression alone. Brief: *"The depression gets you into the target area; the tracer walk gets you on target."*

### Tracer Walk Drill (On Ground Before Departure)

Students should spend 5 minutes in the static cockpit (or paused mission) walking through the concept:

- IP sets a marked point on the ground in the mission; student orbits at low altitude (100 ft AGL) with Master Arm safe; dry-fires (no ammunition loaded) while practicing the head-movement and nose-input technique.
- This reduces cognitive load during the live-fire passes.

## Part 4 — Diving Attack with Minigun

### IP Demonstration

1. IP climbs to 1,200 ft AGL; orbit at 70 KIAS.
2. IP sets depression to 32 mils (80 KIAS, 1,000 m, 10° dive).
3. IP arms; verbal checklist.
4. IP rolls in; establishes 10° dive; accelerates to 80 KIAS.
5. IP begins burst at 1,000 m; talks through tracer walk:
   - *"Rounds are short — I can see tracers striking the ground 50 m in front of the target. I'm lowering the nose fractionally — there — tracers walking toward the target. On target. I'm releasing the trigger and pulling off."*
6. IP demonstrates pull-off; safe sequence verbal.

### Student Passes

**Pass 1 — Diagnostic Pass (No Correction)**
- 10° dive; 80 KIAS; 1,000 m; 32 mils.
- 1-second burst; observe tracer impact.
- Verbalize: *"Tracers were [short/long/left/right] by approximately [distance]."*
- Do not correct — this is the baseline.

**Pass 2 — Correction Pass**
- Use same parameters; apply tracer walk based on Pass 1 observation.
- Walk rounds onto target; 2–3 second burst.

**Pass 3 — Scored Pass**
- All parameters selected and set by student from memory.
- Debrief: IP confirms or corrects the parameter selections.

### Pull-Off Requirements

- Same as rockets: pull-off above 300 ft AGL (10° dive).
- Recoil compensation: the M134's high rate of fire creates a nose-down pitching moment during sustained fire. During the pull-off, account for this by adding slightly more aft cyclic than expected. The nose will pitch down briefly when firing ceases; anticipate and correct.

## Part 5 — Running Fire & Slewing Engagement

### Running Fire with Minigun

Running fire with the minigun follows the same low-altitude profile as with rockets but the interactive tracer walk makes it more forgiving.

**Profile:**
- Altitude: 200–300 ft AGL
- Airspeed: 80 KIAS
- Depression: 35 mils (level fire table)
- Range: 600–800 m

**Procedure:**
1. Arm in transit.
2. Inbound at 200–300 ft AGL; 80 KIAS; nose aligned on target.
3. Begin burst at 800 m; walk tracers onto target.
4. Continue forward; limit pass to 3–4 second burst total.
5. Do not suppress beyond the target — offset 100 m laterally after firing.
6. Safe sequence.

### Slewing to a Lateral Target

The minigun is bore-sighted to the forward axis of the aircraft, so lateral engagements require either a turn or use of the pedal-yaw technique.

**Pedal-yaw technique (same as rockets, Part 3):**
1. Inbound at 200 ft AGL; target is 20° left or right of the nose.
2. Apply pedal to swing the nose; as the sight approaches the target, begin burst.
3. Continue yaw to walk rounds across the target.
4. End burst before nose over-swings; safe sequence.

## Part 6 — Ammo Management & Burst Control

### Why Ammo Management Matters

The M134 fires at approximately 3,000 rpm. At this rate:
- 1 second of fire ≈ 50 rounds
- 3 seconds of fire ≈ 150 rounds per gun
- 5 seconds (max burst) ≈ 250 rounds per gun

With 3,000 rounds per gun, continuous fire would exhaust the load in approximately 60 seconds. In a typical combat engagement, this means:
- **Plan for no more than 10–12 bursts of 3–4 seconds each** before running dry.
- Each pass should accomplish something — don't fire speculatively.

### Ammo State Management

During the range event, IP will call the ammo state at the midpoint:
- *"Ammo state: [X] rounds per gun."*
- Student responds: *"Remaining [X] rounds; approximately [Y] passes remaining at standard burst. Plan: [Z] diving passes, [W] running fire passes."*

### Burst Control Standards

| Burst Type | Duration | Use Case |
|---|---|---|
| Correction burst | 1 second | Diagnostic; first-round observation |
| Standard burst | 3–4 seconds | Effective suppression; standard attack |
| Maximum burst | 4–5 seconds | Sustained suppression; final pass |
| Continuous fire | NOT AUTHORIZED | Training and safety standard; misuse of ammunition |

### Low-Ammo Procedures

When down to the last 500 rounds per gun:

1. IP informs student of the low-ammo state.
2. Student acknowledges: *"Low ammo — switching to SINGLE mode; final two passes; minimum burst."*
3. Student executes no more than 2 additional passes; does not deplete below 100 rounds per gun (DCS may allow fire to zero; procedurally stop at 100).
4. Safe sequence; depart range.

## Common Errors

| Error | Correction |
|---|---|
| Using rocket depression values for minigun (wrong recall) | Brief: the minigun uses the same depression table as rockets at the same speed/range — not a separate table |
| Watching the sight instead of the tracers | Tracer walk is interactive; brief: eyes on the tracer trace, not on the crosshair |
| Over-correcting during tracer walk | Small inputs only (1–2°); over-correction causes rounds to walk off the target in the opposite direction |
| Excessive burst length (>5 seconds) | IP calls stop; re-brief burst discipline and expenditure math |
| Forgetting tracer walk means pulling off late | The temptation is to keep firing until on target; brief: if not on target by 400 ft AGL, pull off — do not press |
| Applying recoil correction after pull-off instead of during | Brief: anticipate the nose-down tendency during the pull; add aft cyclic before firing stops |
