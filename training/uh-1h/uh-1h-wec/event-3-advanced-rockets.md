## Learning Objectives

Upon completion of this event the student will be able to:

- [ ] Select the correct depression value from the three-table system (speed / range / dive angle) without reference to printed tables
- [ ] Conduct a 20° diving attack at 80–100 KIAS; release at 800–1,000 m; pull off above 500 ft AGL
- [ ] Conduct running fire using a pedal turn to track a laterally offset target
- [ ] Fire a WP marking rocket and call the mark to a ground element using standard brevity
- [ ] Identify a danger close scenario and state the required coordination before firing
- [ ] Select and execute PAIRS fire mode; describe the difference between SINGLE, PAIRS, and RIPPLE

## Ground Brief

### Mission Profile

1. Mission planning review: select depression for each attack profile planned this event

3. Ballistic table quiz by IP — student states depression for three called scenarios

5. Part 3: Running fire / pedal turn — 2 passes

7. Part 5: Danger close scenario — IP briefs the scenario; student evaluates and states go/no-go

### Three-Table Memory System

The three tables (by dive angle: level, 10°, 20°) share the same structure. The pattern to remember:

- **More speed** (80 → 100 KIAS): depression **decreases** by ~4 mils (flatter trajectory, longer range)
- **More range** (800 → 1,000 m): depression **increases** by ~6 mils (more gravity drop to compensate)
- **More dive angle** (0° → 10° → 20°): depression **decreases** by ~10 mils per step (dive angle does the pointing work)

If you can remember three anchor values (level/80/800 = 35; 10°/80/800 = 26; 20°/80/800 = 16), you can reconstruct all other values using these deltas.

## Part 2 — 20° Diving Attack

### IP Demonstration

1. IP climbs to 1,500 ft AGL; orbits at 70 KIAS.
2. IP sets depression to 22 mils; arming checklist verbal.
3. IP rolls in to 20° dive; accelerates to 80 KIAS.
4. IP fires at 1,000 m; immediately pulls off — 3 G pull; climb to orbit altitude.
5. IP states: *"A 20° dive is steeper — the nose is more depressed at roll-in. The aircraft does more of the pointing work, so you need less sight depression. The tradeoff: you need more altitude to enter, and your minimum recovery altitude is 500 ft, not 300 ft."*
6. IP highlights: *"At 20°, you will feel a greater tendency to over-dive — the target can disappear under the nose. Keep the target in sight the entire time. If you lose the target, abort the pass."*

### Student Passes

**Pass 1 — 20° Dive, 80 KIAS, 1,000 m, 22 mils**

1. Climb to 1,400–1,600 ft AGL; orbit at 70 KIAS.
2. Depression: 22 mils.
3. Arming checklist — verbal, before roll-in.
4. Roll-in: establish 20° nose-low; accelerate to 80 KIAS.
5. Track target in sight; fire at ~1,000 m slant range.
6. Pull-off: 3 G pull; climb; break.
7. **Altitude floor: 500 ft AGL.** Do not press below this altitude.
8. Verbalize impact; safe sequence.

**Pass 2 — 20° Dive, 80 KIAS, 800 m, 16 mils**

Same profile; change depression to 16 mils; fire at shorter range.

**Pass 3 — 20° Dive, 100 KIAS, 1,000 m, 18 mils**

Set depression to 18 mils; stabilize at 100 KIAS before roll-in.

### 20° vs. 10° Comparison (IP Debrief Point)

After the three passes:

| Parameter | 10° Dive | 20° Dive |
|---|---|---|
| Entry altitude | 1,000–1,200 ft | 1,400–1,600 ft |
| Depression at 800 m / 80 KIAS | 26 mils | 16 mils |
| Min recovery altitude | 300 ft AGL | 500 ft AGL |
| Target tracking difficulty | Moderate | High (target disappears under nose) |
| Accuracy potential | Moderate | Higher (steeper = less drift sensitivity) |
| Use case | General support fire | Pinpoint targets; harder surfaces |

## Part 3 — Running Fire & Pedal Turn Techniques

### Running Fire

Running fire is conducted when the target is approximately level with the aircraft (close to the same altitude), the aircraft is flying a path that passes adjacent to or over the target, and the geometry does not allow a classic diving attack.

**Profile:**
- Altitude: 200–300 ft AGL
- Airspeed: 80–90 KIAS
- Depression: 35 mils (same as level fire)
- Attack heading: directly toward target, or at a shallow offset angle (≤15°)

**Procedure:**

1. Arm in the cruise before entering the target zone.
2. On the inbound leg, maintain 200–300 ft AGL; 80 KIAS.
3. Align nose on target; track with depression set.
4. Fire 1–2 rockets when sight is on target; observe impact.
5. Continue through; safe sequence; re-attack or depart.

### Pedal Turn Technique

Used when the target is positioned to the side and a full orbit is not possible or efficient. The pilot uses left/right yaw pedal to swing the nose laterally to acquire a target in the forward arc without banking.

**Procedure:**

1. Inbound at 200–300 ft AGL; 60–70 KIAS.
2. Target is offset 15–25° left or right of the nose.
3. Apply yaw pedal (left pedal = nose swings left; right pedal = nose swings right) to align sight.
4. As nose approaches alignment, reduce pedal; fire when sight crosses target.
5. Return to straight flight; safe sequence.

> **Note:** In DCS, the yaw pedals are assigned to the rudder axis. The UH-1H's tail rotor is sufficiently powerful to swing the nose 20–30° in a few seconds at low airspeeds. Practice this technique at hover and slow flight before applying it in running fire.

### Student Passes (Running Fire)

- 2 passes at 200–300 ft AGL; 80 KIAS; depression 35 mils.
- 1 pass using pedal turn to track a 20° offset target.

## Part 4 — WP (White Phosphorus) Marking Rockets

### Purpose of WP Rockets

M156 White Phosphorus rockets are used to mark target locations for other aircraft, for ground elements who need visual orientation, or to set fires. The round produces a large white smoke cloud on impact.

### DCS Loadout Note

The M156 WP warhead is available in the mission editor for the XM-158 and XM-159 pods. Select "WP" warhead type when loading. In DCS, WP impact produces a visible white smoke effect that persists for approximately 90 seconds.

### Firing a Marking Rocket

**Profile:**
- Use 10° dive profile; 80 KIAS; 1,000 m range; 32 mils depression.
- Fire 1 rocket only (SINGLE mode).
- Observe impact.

**Standard Mark Call (after impact):**

> *"[Callsign], mark is [direction of smoke from reference point] at [distance]. Target [description] is [direction and distance from smoke]."*

**Example:**

> *"Warlord 6, mark is 200m north of the tree line, smoke is white. Target armored vehicle is 50m east of smoke."*

### Student Execution

1. IP designates a target point on the range.
2. Student loads WP in the fire mode selector position.
3. Student arms; fires 1 WP rocket on 10° dive; observes impact.
4. Student makes the standard mark call (IP simulates ground element response).
5. Student identifies target relative to smoke.

### Danger Close Reminder for WP

M156 WP danger close distance: **300 m**. WP is NOT to be fired within 300 m of friendly positions even as a marker — the burning phosphorus creates a larger hazard radius than a standard HE round. Always confirm friendly positions before firing WP.

## Part 5 — Danger Close Engagement

### Defining Danger Close

A "danger close" engagement is one where the target is within the established minimum safe distance of friendly forces. These distances represent the minimum range at which the cited munition is considered safe (for troops in the open) with reasonable assurance.

| Munition | Danger Close Distance |
|---|---|
| M151 / M229 HE rockets | 230 m from friendly |
| M156 WP marking rockets | 300 m from friendly |
| M134 Minigun (7.62mm) | 200 m from friendly |
| M60D Door Gun (7.62mm) | 200 m from friendly |

### Scenario (IP Briefs)

> *"Warlord 6, I have friendlies at grid [X]. The target is at grid [Y]. Your AO is bounded north by the treeline. Request rockets on the enemy position. Friendlies are in covered positions. Over."*

Student calculates or estimates the separation distance. If the target–friendly distance is less than the listed danger close distance for the munition to be used:

- **State:** "This is danger close."
- **Action:** "Requesting clearance with modified parameters — I will use SINGLE fire, reduce the number of rockets, and confirm friendly positions before engaging."
- If cleared danger close: increase accuracy requirements; fire SINGLE; observer must confirm rounds clear of friendlies after each impact.
- If not cleared: decline the engagement and recommend standoff weapons or a different attack vector.

### Danger Close Drill

IP runs student through three danger close go/no-go scenarios verbally:

| Scenario | Friendly–Target Distance | Munition | Decision |
|---|---|---|---|
| A | 250 m | M151 HE rocket | Danger close — state and coordinate |
| B | 350 m | M156 WP | Danger close — WP is 300 m |
| C | 220 m | M134 minigun | Danger close — 200 m limit |
| D | 300 m | M151 HE rocket | Clear — 300 m > 230 m |
| E | 320 m | M156 WP | Danger close — WP is 300 m |

Student must correctly identify all five before the flight event is complete.

## Part 6 — Salvo & Ripple Fire

### Fire Mode Overview

The armament panel intervalometer controls how rockets fire.

| Mode | Behavior | Use Case |
|---|---|---|
| **SINGLE** | One rocket per trigger press | Observation; conservation; all training passes |
| **PAIRS** | Two rockets per trigger press (one per pylon) | Standard combat fire; area suppression |
| **RIPPLE** | Continuous fire while trigger held; rockets fire in sequence | Area saturation; maximum volume |

### DCS Implementation

- In DCS, fire mode is set via the intervalometer selector on the left cockpit panel.
- The DCS UH-1H models SINGLE and PAIRS (and in some configurations RIPPLE as continuous fire while trigger held).
- For all training passes in this course, default to SINGLE unless the IP specifies otherwise.

### Student Demonstration

1. IP selects a target area; loads a full pod (14 rockets minimum).
2. Student sets PAIRS mode; executes one diving pass; fires 2 trigger presses (4 rockets total).
3. Student observes the impact spread — PAIRS fire lands as two closely spaced impacts per press.
4. Student states: *"In PAIRS mode, two rockets fired simultaneously — I can see two impacts per trigger press. This is more effective for area targets but uses ammunition faster."*
5. IP transitions student back to SINGLE for all subsequent passes.

## Common Errors

| Error | Correction |
|---|---|
| Applying 10° depression values on 20° attack | Review: steeper dive = less depression; brief the anchor value table |
| Pressing below 500 ft AGL on 20° dive | 500 ft is the hard floor; IP intervenes at 600 ft |
| Losing target under nose at 20° dive | Brief: if target disappears, abort and re-orbit; do not chase the target by steepening further |
| WP mark call missing target orientation | Practice the call format: smoke, then target from smoke |
| Evaluating danger close using wrong munition distance | Each munition has its own distance; WP is 300 m (not 230 m); minigun is 200 m |
| Setting PAIRS or RIPPLE for observation passes | Default is SINGLE unless IP directs otherwise; mention it every pass |
