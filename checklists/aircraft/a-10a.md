## Performance & V-Speeds

| Speed | Light (< 35K lbs) | Medium (35–42K lbs) | Heavy (> 42K lbs) |
|---|---|---|---|
| **Vr (Rotation)** | ~105 KIAS | ~115 KIAS | ~130 KIAS |
| **V2 (OEI Safety)** | ~120 KIAS | ~130 KIAS | ~145 KIAS |
| **Vmca (Min Control)** | ~110 KIAS | ~115 KIAS | ~120 KIAS |
| **Vref (Final Approach)** | ~120 KIAS | ~130 KIAS | ~140 KIAS |
| **Best Glide** | ~200 KIAS | ~200 KIAS | ~210 KIAS |
| **Gear/Flap Limit** | 200 KIAS | 200 KIAS | 200 KIAS |
| **Cruise** | 250–280 KIAS | 240–270 KIAS | 220–250 KIAS |

> **Approach speed note:** Always fly final approach by **AOA indexer** (green donut = on-speed) — not by a fixed airspeed. The correct speed changes with weight; the indexer is always accurate.

> **Glide ratio:** ~8–10:1 clean. From 10,000 ft AGL with both engines failed, expect approximately 13–17 NM glide range.

## AOA Indexer Reference

The AOA indexer is mounted on the canopy frame (left side) and overlaid on the HUD. It is the primary reference for all approach speeds.

| Indicator | Meaning | Action |
|---|---|---|
| **○ (Green donut)** | On-speed — correct AoA | Maintain |
| **△ (Amber UP chevron)** | Slow / high AoA | Add power; lower nose slightly |
| **▽ (Amber DOWN chevron)** | Fast / low AoA | Reduce power; allow pitch to rise |
| **△ + ○** | Slightly slow | Small power addition |
| **▽ + ○** | Slightly fast | Small power reduction |

## Startup Checklist

### Phase 1 — Electrical Power

- [ ] **Electrical Power** — ON (`RShift + L`)
  - HUD activates; avionics power up

### Phase 2 — Left Engine Start

- [ ] **Throttle** — IDLE position (both)
- [ ] **Left Engine Start** — `RAlt + Home`
  - N2 begins rising; EGT rises then settles
  - **EGT must stay below 725°C** — if exceeded, hot start → stop immediately (`RAlt + End`)
- [ ] Left engine at ground idle — confirm N2 ~55–60%, EGT < 650°C

### Phase 3 — Right Engine Start

- [ ] **Right Engine Start** — `RCtrl + Home`
  - Same monitoring — EGT below 725°C during start
- [ ] Right engine at ground idle — confirm N2 ~55–60%, EGT < 650°C

### Phase 4 — Post-Start Checks

- [ ] Both engines — N2 stable ~55–60%, EGT < 650°C, all cautions clear
- [ ] **Altimeter** — set QNH (`RAlt + Num+` / `RAlt + Num-`); match field elevation
- [ ] **Navigation** — available immediately in FC3 (no INS alignment required)
- [ ] **Flight controls** — check full deflection all axes (`F2` external view to confirm)
- [ ] **Canopy** — closed and locked (`LCtrl + C`)

> **Hot Start Recovery:** If EGT exceeds 725°C during start — immediately press engine stop (`RAlt + End` left / `RCtrl + End` right). Wait 30–60 seconds, then reattempt.

> **FC3 Startup Note:** Real A-10A startup requires APU/JFS operation, INS alignment (4–8 minutes), and extensive cockpit panel sequencing. DCS FC3 abstracts this to one keybind per engine. EGT monitoring behaviour is still realistic.

## Taxi & Before-Takeoff

### Taxi

- [ ] **Parking Brake** — RELEASE (`LCtrl + W` to toggle off)
- [ ] **Taxi lights** — ON (`L`)
- [ ] Advance throttle slightly — aircraft will begin rolling
- [ ] **Brakes** — test immediately (`W` — both; verify function)
- [ ] **Taxi speed** — 10–15 knots; do not exceed 20 knots
- [ ] **Nosewheel steering** — rudder pedals / rudder axis; ±45° authority
- [ ] **Tight turns** — use differential braking to assist NWS
- [ ] **Wingtip clearance** — wingspan 57.5 ft; watch obstacles

> **NWS note:** The A-10A has nosewheel steering controlled by rudder pedals/axis. It is active automatically on the ground and provides ±45° of authority. Differential braking is available for tighter turns.

### Before-Takeoff Checks (Hold-Short Line)

- [ ] **Brakes** — HOLD
- [ ] **Flight Controls** — full deflection all axes; confirm surface movement
- [ ] **Trim** — neutral or slight nose-up for takeoff
- [ ] **Flaps** — set as required
  - Normal takeoff: UP
  - Short-field: MVR (`LCtrl + F` one press from UP)
- [ ] **Speed Brakes** — CLOSED (`LCtrl + B`)
- [ ] **Altimeter** — set and confirmed
- [ ] **Fuel** — check quantity; confirm sufficient for mission
- [ ] **Canopy** — CLOSED (`LCtrl + C`)
- [ ] **Engine instruments** — both N2 ~55–60%, EGT < 650°C

## Takeoff

### Normal Takeoff (Flaps UP)

- [ ] **Line up** on runway centerline; align nosewheel
- [ ] **Brakes** — HOLD (`W`)
- [ ] **Throttle** — advance to ~85% N2; hold 2–3 seconds; verify both engines match
- [ ] **Release brakes** — aircraft begins rolling
- [ ] **Throttle** — smoothly to MIL (100% N2)
- [ ] **Track centerline** — rudder pedals; aileron into any crosswind
- [ ] **Rotate** — at Vr (weight-dependent: ~105–130 KIAS); smooth 2–3°/sec pitch rate to 8–10° nose-up
- [ ] **Gear UP** — positive climb rate + no runway remaining → `G`
- [ ] **Climb** — maintain 8–10° pitch; accelerate to 150–160 KIAS initial climb

### Short-Field Takeoff (MVR Flaps)

- [ ] **Flaps** — MVR (`LCtrl + F` one press)
- [ ] **Hold brakes** — throttle to MIL while stationary
- [ ] **Release brakes** — rolling start at full thrust
- [ ] **Rotate** — same Vr as normal (lifts off ~5–8 KIAS sooner with flaps)
- [ ] **Gear UP** — positive rate, no runway remaining
- [ ] **Accelerate** to ≥ 150 KIAS, then **Flaps UP** (`LShift + F`)

### Takeoff Abort (Rejected Takeoff)

| Decision Point | Action |
|---|---|
| **Below Vr** | Throttle IDLE → maximum brakes (`W`) → speed brakes open (`LShift + B`) → maintain centerline |
| **Above Vr, engine failure** | If runway remains: land straight ahead. If airborne: continue to V2 → single-engine procedure |

**Abort criteria:** Engine failure, caution light, directional control loss, unusual vibration, any doubt.

## Basic Flight & Airwork

### Level Flight & Trim

- [ ] Target cruise speed per weight class (see V-Speeds table)
- [ ] **Trim** — re-trim after every power, speed, or configuration change
  - Pitch: `RCtrl + ;` (up) / `RCtrl + .` (down)
  - Roll: `RCtrl + ,` (left) / `RCtrl + /` (right)
  - Rudder: `RCtrl + Z` (left) / `RCtrl + X` (right)
- [ ] "If holding steady pressure for more than 5 seconds — trim it out"
- [ ] After weapons release: expect roll toward the heavy wing; retrim immediately

### Turns

| Bank Angle | Application |
|---|---|
| 15–20° | Gentle maneuvering; heading corrections |
| 30° | Standard turns; pattern work |
| 45° | Steep turn (training) — add ~5% N2; back-pressure to maintain altitude |
| 60° | ~2.0 G loading; significant back-pressure required |

### Steep Turn (45° Bank) — Standards

1. Clear area; establish 250 KIAS at entry altitude
2. Roll smoothly to 45°; add slight back-pressure and ~5% N2
3. Maintain altitude ±100 ft, airspeed ±10 KIAS through the turn
4. Begin rollout ~30° before desired heading
5. Reduce power, re-trim on exit

### Stall Recovery

**Clean configuration stall (~130 KIAS light weight):**
1. Buffet onset at ~18–20 AoA units — AOA indexer shows amber UP chevron
2. Full stall at ~23–25 AoA — nose drops; possible wing rock if uncoordinated
3. Recovery: **Power MIL → lower nose below horizon → wings level → recover**

**Dirty configuration stall (~110 KIAS light weight):**
1. Same recovery: **Power MIL → lower nose → wings level → gear up → flaps up incrementally**

> **Stall character:** The A-10A straight wing gives ample warning before stall break. The aircraft does not snap or depart if controls are coordinated. Spin entry requires deliberate effort.

### Speed Limits Summary

| Configuration | Max Speed |
|---|---|
| **Clean** | 450 KIAS / Mach 0.75 |
| **Gear extended** | 200 KIAS |
| **Flaps (MVR or DN)** | 200 KIAS |
| **Speed brakes** | No restriction |

## Traffic Pattern

### Overhead Pattern (Break Pattern) — Standard USAF

```
       ← Initial (250–300 KIAS, 1,500 ft AGL)
  ┌─────────────────────────────┐
  │   OVERHEAD BREAK            │
  │   (break at approach end)   │
  │                             │
  │  Downwind (150–170 KIAS)    │
  ◄──────────────────────────── │
  │   Gear DN, Flaps as req'd   │
  │                             │
  │     Base (130–140 KIAS)     │
  │     ┌───────────────┐       │
  │     │               │       │
  │     │    Final      │       │
  │     │    (Vref)     │       │
  └─────┘               │       │
  ════════[RUNWAY]═══════       │
  ▲                             │
  └─────────────────────────────┘
```

| Leg | Altitude | Speed | Configuration | Actions |
|---|---|---|---|---|
| **Initial** | 1,500 ft AGL | 250–300 KIAS | Clean | ATC contact; request overhead entry |
| **Break** | 1,500 ft AGL | 250+ KIAS | Clean → transitioning | Aggressive turn (60–70° bank); power to idle; decelerate |
| **Downwind** | 1,500 ft AGL | 150–170 KIAS | Gear DOWN below 200 KIAS (`G`); Flaps MVR or DN | Check 3 green; verify gear down |
| **Base** | ~1,000 ft AGL | 130–140 KIAS | Dirty | Begin descent; align for final |
| **Final** | Descending | Vref (AOA donut) | Dirty | Stabilized by 300 ft AGL |

### Rectangular Pattern

| Leg | Altitude | Speed | Notes |
|---|---|---|---|
| **Upwind/Departure** | Climb to 1,500 ft | 150+ KIAS | Climb to pattern altitude |
| **Crosswind** | 1,500 ft | 170–180 KIAS | Level; begin configuring |
| **Downwind** | 1,500 ft | 150–170 KIAS | Gear down; flaps as required |
| **Base** | Descending | 130–140 KIAS | Descend; align for final |
| **Final** | Descending | Vref (AOA donut) | Stabilized approach |

## Approach & Landing

### Stabilized Approach Criteria (300 ft AGL gate)

By 300 ft AGL, ALL of the following must be met — if not, **GO AROUND**:

- [ ] **On-speed** — AOA indexer showing green donut (or within ±5 KIAS of Vref)
- [ ] **On glidepath** — ~3° descent; neither ballooning nor sinking
- [ ] **On centerline** — aligned with runway
- [ ] **Configuration set** — gear down (3 green), flaps as required
- [ ] **Power stable** — no large throttle corrections needed

### Normal Landing (Flaps DN)

- [ ] **Downwind** — 1,500 ft AGL; 150–170 KIAS; Gear DOWN (`G`); Flaps DN (`LCtrl + F` × 2)
- [ ] **Confirm** — 3 green; gear down and locked
- [ ] **Base** — begin descent; 130–140 KIAS; ~30° bank turn
- [ ] **Final** — Vref speed; AOA indexer **green donut**; ~3° glidepath
- [ ] **300 ft gate** — stabilized check (see above); if not met → GO AROUND
- [ ] **Threshold** — cross ~50 ft AGL; begin reducing power
- [ ] **Flare** — gentle 2–3° pitch increase; smooth power to idle
- [ ] **Touchdown** — main gear first; hold nose off briefly
- [ ] **Nosewheel** — lower gently with back-stick; do not slam down
- [ ] **Decelerate** — brakes (`W`); speed brakes open (`LShift + B`)
- [ ] **Exit runway** — at nearest taxiway; report clear

### Short-Field Landing

- [ ] **Approach speed** — Vref (no addition; minimum safe speed)
- [ ] **Aim point** — ~500 ft from threshold
- [ ] **Touchdown** — firm; on the aim point; do not float
- [ ] **Decelerate** — maximum braking immediately; speed brakes open

### Crosswind Landing

| Crosswind | Technique |
|---|---|
| 0–10 kts | Normal; slight wing-low into wind on short final |
| 10–20 kts | Pronounced wing-low; de-crab at threshold; use opposite rudder |
| 20–25 kts | Maximum demonstrated; consider alternate runway |
| > 25 kts | Do not attempt; divert |

**Crosswind technique:** Crab on final → transition to sideslip at ~200 ft AGL (lower upwind wing, opposite rudder) → upwind main gear touches first → hold centerline with rudder through rollout.

### Single-Engine Approach

- [ ] **Approach speed** — Vref + 10 KIAS (additional go-around margin)
- [ ] **Pattern** — wide, flat; avoid steep turns toward the dead engine
- [ ] **Flaps** — MVR (7°) preferred; reduces stall speed without excessive drag
- [ ] **Plan to land first approach** — go-around is marginal; do not assume it is available
- [ ] **Touchdown** — firm; accept it; do not float or attempt precision flare

> **Single-engine go-around:** Only if above Vmca + 10 KIAS and above 300 ft AGL. Below Vmca or below 200 ft AGL — **commit to landing.**

### No-Flap Landing

- [ ] **Approach speed** — Vref + 15 KIAS (higher stall speed without flaps)
- [ ] **Glidepath** — flatter (less drag = shallower approach)
- [ ] **Landing distance** — significantly increased; confirm adequate runway length

## Go-Around

**GO AROUND IS MANDATORY** if: not stabilized by 300 ft AGL / runway not clear / excessive sink / speed deviation ≥ 10 KIAS / any doubt.

- [ ] Announce: *"[Callsign], going around"*
- [ ] **Power** — FULL MIL (`Num +`) — immediately, no hesitation
- [ ] **Pitch** — 8–10° nose up; positive rate of climb
- [ ] **Gear** — UP (`G`) — once positive rate confirmed
- [ ] **Speed Brakes** — CLOSE (`LCtrl + B`) — if extended
- [ ] **Flaps** — retract incrementally (`LShift + F`) — MVR first, then UP above 150 KIAS
- [ ] **Climb** — accelerate to 150–160 KIAS; re-enter pattern or depart

## Shutdown Checklist

- [ ] **Taxi** to parking at ≤ 15 knots
- [ ] **Parking Brake** — SET (`LCtrl + W`)
- [ ] **Throttle** — IDLE; allow 2-minute cool-down at idle
- [ ] **Left Engine** — STOP (`RAlt + End`)
- [ ] **Right Engine** — STOP (`RCtrl + End`)
- [ ] **Electrical Power** — OFF (`RShift + L`)
- [ ] **Canopy** — OPEN (`LCtrl + C`) for ground crew access

## Emergency Procedures

### Boldface — ENGINE FIRE (Single Engine)

```
1. THROTTLE (affected engine) ........ IDLE
2. ENGINE (affected engine) .......... SHUT DOWN
   Left:  RAlt + End
   Right: RCtrl + End
3. FIRE indication ................... MONITOR
   • Fire extinguishes → proceed to nearest airfield
   • Fire persists → PREPARE TO EJECT
4. SINGLE-ENGINE procedures .......... EXECUTE
```

### Boldface — DUAL ENGINE FAILURE

```
1. AIRSPEED .......................... MAINTAIN ~200 KIAS (best glide)
   Glide ratio ~8–10:1 clean; from 10,000 ft ≈ 13–17 NM range
2. RESTART ATTEMPT ................... EXECUTE
   Left Engine:  RAlt + Home
   Right Engine: RCtrl + Home
3. IF restart fails .................. PREPARE FOR FORCED LANDING
   • Aim for longest available runway or flat terrain
   • Gear DOWN below 200 KIAS (G)
4. IF no landing site available ...... EJECT (LCtrl + E × 3)
```

### Boldface — DUAL HYDRAULIC FAILURE / MANUAL REVERSION

```
1. AIRSPEED .......................... REDUCE TO ≤ 200 KIAS
2. PITCH TRIM ........................ USE EXTENSIVELY
   (electric trim still functions — primary pitch control aid)
3. CONTROL INPUTS .................... SMOOTH, SMALL, DELIBERATE
   (reduced authority; anticipate response lag)
4. CONFIGURE EARLY ................... GEAR AND FLAPS WELL BEFORE FINAL
5. PLAN LONG STRAIGHT-IN APPROACH ... 10+ MILES IF POSSIBLE
```

### Single-Engine Operations

| Situation | Action |
|---|---|
| **Engine failure in cruise** | Identify → shut down if fire/oil failure → trim → nearest airfield |
| **Engine failure in pattern** | Downwind/base: continue and land. Initial approach: go-around if above Vmca. Below Vmca: lower nose immediately. |
| **Engine fire** | Shut down immediately; if fire persists > 30 seconds → prepare to eject |

**Single-engine flight:**
- [ ] Maintain ≥ Vmca (~110–120 KIAS, weight dependent) — **NEVER below this**
- [ ] Significant yaw toward dead engine — correct with rudder immediately
- [ ] **Rudder trim** toward good engine — essential
- [ ] Clean up: gear up, flaps up, speed brakes closed
- [ ] Plan for nearest suitable airfield

### Emergency Jettison

- [ ] **Emergency Jettison (All Stores)** — `LAlt + R`

> ⚠ **Keybind warning:** `LCtrl + W` is the **parking brake** — NOT emergency jettison. Use `LAlt + R` for emergency jettison.

### Ejection

- [ ] **Conditions:** Aircraft uncontrollable below 10,000 ft AGL; dual engine failure with no landing site; unextinguishable fire; structural failure
- [ ] **Decide early** — do not delay below 2,000 ft AGL
- [ ] **Eject** — `LCtrl + E` × **3 presses** (triple-press to prevent accidental ejection)

> **ACES II seat is zero-zero capable** — can eject from 0 altitude / 0 airspeed. Higher altitude = more time = more options.

### Landing Gear Malfunction

- [ ] **Gear fails to extend** — recycle: gear up (`G`), wait, gear down (`G`)
- [ ] **Still fails** — plan gear-up (belly) landing: minimum safe speed, long flat approach
- [ ] **Belly landing** — accept aircraft damage; DCS models gear-up landing survivably

### Electrical Failure

- [ ] HUD blanks / instruments lost → fly by external references
- [ ] Attempt reset: `RShift + L` (toggle power)
- [ ] **Land ASAP** — battery life is limited
