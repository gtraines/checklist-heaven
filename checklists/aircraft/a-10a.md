## Cockpit Orientation

The A-10A uses conventional analog cockpit instruments with limited avionics compared to the A-10C.

| Panel | Location | Function |
|---|---|---|
| HUD | Forward | Flight data, weapon cues |
| Weapon Management Panel | Center | Select and arm stations |
| RWR (ALR-45/56) | Lower right | Radar warning |
| ECM Panel | Right console | Electronic countermeasures |
| Radio (ARC-164 UHF) | Right console | Primary radio |
| Radio (ARC-186 VHF) | Left console | Secondary radio |

## Startup Checklist

### Pre-Start
- [ ] **Parking Brake** — SET (`W`)
- [ ] **Throttle** — IDLE
- [ ] **Battery Switch** — ON
- [ ] **Emergency Flight Controls** — check normal
- [ ] **Fuel** — check quantity and feed switches

### Engine Start (TF34-GE-100A × 2)
- [ ] **Left Engine Start** — `RAlt + Home`
  - EGT rising — normal
  - N1 > 55% — throttle to IDLE
- [ ] **Right Engine Start** — `RCtrl + Home`
  - N1 > 55% — throttle to IDLE
- [ ] Both engines at IDLE — confirm stable
- [ ] **Generators** — check both ON
- [ ] **Hydraulics** — check A and B systems pressure
- [ ] **Flight controls** — check freedom of movement

### Avionics Warmup
- [ ] **UHF Radio (ARC-164)** — ON, preset channel or manual freq
- [ ] **VHF Radio (ARC-186)** — ON, set frequency
- [ ] **RWR (ALR-45)** — ON
- [ ] **ECM pod** — ON if carried
- [ ] **Weapon systems** — power ON
- [ ] Allow ~2 min for systems to warm up

## Taxi & Takeoff

### Taxi
- [ ] **Parking Brake** — RELEASE (`W`)
- [ ] **Taxi speed** — 10–20 kph; use differential thrust
- [ ] No nose-wheel steering — use differential braking and thrust

### Takeoff
- [ ] **Flaps** — check (no flaps needed for A-10A standard TO)
- [ ] **Throttle** — FULL (`PgUp`)
- [ ] Rotate at ~140 kts (260 kph)
- [ ] **Gear** — UP (`G`) at positive climb rate
- [ ] Climb at 200–220 kts, 15°–20° nose up

## Navigation

### Heading Reference Navigation
The A-10A uses conventional navigation without GPS.

- [ ] **Compass heading** — set before takeoff
- [ ] **TACAN** — set station if available
- [ ] **VOR/ILS** — set frequency if available
- [ ] **Dead reckoning** — use time, speed, heading
- [ ] **F10 Map** — use for situational awareness (`F10`)

### TACAN Navigation
- [ ] **TACAN channel** — set on instrument
- [ ] **Mode** — T/R (transmit/receive) or A/A
- [ ] **CDI needle** — center for bearing to station
- [ ] **DME** — read distance to station

## Weapons Employment

### Master Arm & Safety
- [ ] **Master Arm Switch** — SAFE during transit
- [ ] **Master Arm Switch** — ARM (`O` key) in attack phase
- [ ] **Guns Arm** — required separately for GAU-8

### GAU-8/A 30mm Cannon

#### Description
- 1,174 rounds standard load
- Rate of fire: 3,900 or 4,200 rpm
- Very effective against all ground targets including light armor

#### Strafing Run
1. [ ] **Master Arm** — ARM
2. [ ] **Gun** — select
3. [ ] Dive: 10°–20° angle, wings level
4. [ ] **Line up** 2,000 m from target
5. [ ] **Open fire** at 1,200–1,500 m
6. [ ] **Cease fire** at 500 m
7. [ ] Burst: 1–3 seconds maximum
8. [ ] **Pull off** with 3–4G, minimum 600 m AGL
9. [ ] **Fire:** `Space` or trigger

> **Note:** The GAU-8 is extremely effective in shallow dives. Multiple short bursts more accurate than one long burst.

### Rockets

#### 2.75-inch FFAR / 5-inch Zuni
1. [ ] **Master Arm** — ARM
2. [ ] Select rocket pods on weapon panel
3. [ ] Set salvo quantity
4. [ ] Dive 15°–30°
5. [ ] Fire at 1,200–2,000 m range
6. [ ] Pull off at 800 m minimum

### Unguided Bombs

#### Mk-82 / Mk-83 / Mk-84 General Purpose
- **Mk-82:** 500 lb, general purpose
- **Mk-83:** 1,000 lb
- **Mk-84:** 2,000 lb

#### Retarded Bombs (Mk-82 AIR, Mk-82 Snakeye)
- Fin opening slows bomb for low-level delivery

#### CCIP Delivery
1. [ ] **Master Arm** — ARM
2. [ ] Select bombs on weapon panel
3. [ ] Dive 20°–45°
4. [ ] Align CCIP pipper on target (HUD)
5. [ ] **Release** when pipper reaches target: `Space`
6. [ ] Pull off: maintain 600 m AGL minimum (retarded) / 1,500 m (slick)

#### Level/Loft Delivery
1. [ ] Level flight at planned altitude
2. [ ] **Auto-release** via HUD when computed release point reached
3. [ ] Useful for stand-off delivery

### AGM-65 Maverick

The AGM-65 is the A-10's primary precision air-to-ground missile.

| Variant | Seeker | Notes |
|---|---|---|
| AGM-65A/B | TV (Electro-Optical) | Daylight, good weather |
| AGM-65D | IIR (Infrared Imaging) | Day/Night, all weather |
| AGM-65G | IIR, large warhead | Tanks, heavy armor |

#### AGM-65 Employment
1. [ ] **Master Arm** — ARM
2. [ ] Select AGM-65 station on weapon panel
3. [ ] **Maverick power** — ON (allow 3–5 min cooldown for IIR variants)
4. [ ] **Seeker** — uncage and slew to target area
   - Slew: `Numpad 8/2/4/6`
5. [ ] **Lock** on target: `Numpad Enter`
6. [ ] Confirm tone change (acquisition tone)
7. [ ] **Launch envelope:** 500 m – 8 km
8. [ ] **Fire:** `Space` or trigger
9. [ ] **Fire-and-forget** — break off after launch

> **Note:** Lock confirmed by seeker squaring on target in MFD/HUD

### Cluster Munitions

#### CBU-52 / CBU-87 / CBU-97
1. [ ] **Master Arm** — ARM
2. [ ] Select CBU on weapon panel
3. [ ] Set fuze settings (altitude of burst)
4. [ ] Level delivery: altitude 300–600 m AGL for typical burst
5. [ ] **Release** in level flight or shallow dive: `Space`
6. [ ] Maintain pattern — do not overfly burst area immediately

## Defensive Systems

### ALR-45/56 RWR
- Provides radar threat detection and classification
- Audio and visual alerts on threat panel

### ALE-40 Chaff/Flare Dispensers
- [ ] **Chaff** — `[` key
- [ ] **Flares** — `]` key
- [ ] **Program** — set chaff/flare program via dispenser control panel
- [ ] On SAM launch — break + flares + chaff

### ALQ-131 ECM Pod (if equipped)
- [ ] Power ON before entering threat area
- [ ] Provides standoff jamming against radar-guided systems

### Defensive Break
1. Hard turn into threat (3–5G)
2. Flares: 2–4 bursts
3. Chaff: 2 bursts
4. Dive for terrain/trees masking
5. Extend from threat

## Landing Checklist

- [ ] **Fuel** — check (> 500 lbs remaining)
- [ ] **Speed** — reduce to 200 kts on downwind
- [ ] **Gear** — DOWN (`G`) — 3 green lights
- [ ] **Approach speed** — 150 kts (clean), 140 kts with flaps
- [ ] **Touchdown** — 120 kts
- [ ] **Brakes** — `W`
- [ ] **Reverse thrust** — N/A (TF34 not reversible in DCS A-10A)

## Shutdown Checklist

- [ ] **Taxi** to parking
- [ ] **Parking Brake** — SET (`W`)
- [ ] **Throttle** — IDLE then CUT-OFF (`PgDn` full)
- [ ] Wait for engine spool-down
- [ ] **Avionics** — all OFF
- [ ] **Battery** — OFF

## Emergency Procedures

### Single Engine
1. Dead engine — throttle to IDLE / OFF
2. Maintain directional control with rudder
3. Reduce drag — check stores (jettison if needed: `LAlt + J`)
4. Single-engine approach: +15 kts approach speed

### Hydraulics Failure
1. Emergency gear — `LCtrl + G`
2. Flapless approach
3. Add 10–15 kts to approach speed

### Battle Damage Assessment
- The A-10 is designed to absorb tremendous damage
- If control degraded — reduce bank angle, minimize G loading
- Fly toward friendly territory before deciding to eject

### Ejection
- `LCtrl + E` — hold 1 second
- Min safe: 100 m AGL
