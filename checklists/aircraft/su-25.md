## Cockpit Orientation

The Su-25 (Frogfoot-A) is a simplified version of the Su-25T, without the Shkval targeting system. It uses a basic HUD and conventional weapons delivery.

| Panel | Location | Function |
|---|---|---|
| HUD | Forward | Flight data, weapon delivery cues |
| SPO-15 RWR | Right console | Radar Warning Receiver |
| Weapon panel | Right console | Weapons selection and arming |
| Radio panel | Left console | R-862 radio |
| Navigation panel | Center | Heading/Waypoint |

## Operating Limitations

| Parameter | Limit | Notes |
|---|---|---|
| **Max Takeoff Weight (MTOW)** | ~17,600 kg | Performance degrades significantly > 14,600 kg |
| **Max Landing Weight** | 13,300 kg | Risk of gear damage if exceeded (> 5 m/s sink) |
| **G-Limits** | +6.5 / -2.0 (Clean)<br>+5.0 (Loaded) | Watch G-meter; asymmetric loads reduce limit |
| **AoA Limits** | ~18° (Stall Warning)<br>~20° (Departure) | Stall buffet provides tactile warning |
| **Crosswind Component** | 12-15 m/s | 12 m/s limit in manual, 15 m/s achievable |
| **Max Mach** | M 0.82 | Airframe vibration/buffet near limit |

## Emergency Procedures (Quick Ref)

| Emergency | Immediate Action Steps |
|---|---|
| **Engine Failure on Takeoff** | 1. **Abort** (if < V1) / **Jettison** (`LCtrl + W`)<br>2. **Rudder** — Maintain Direction<br>3. **Speed** — > 300 km/h |
| **Engine Fire in Flight** | 1. **Throttle** — IDLE (Affected)<br>2. **Stopcock** — OFF (`RAlt/RCtrl + End`)<br>3. **Land ASAP** |
| **Engine Restart** | 1. **Alt/Speed** — < 6km, 400-600 km/h<br>2. **Throttle** — IDLE<br>3. **Start** — (`RAlt/RCtrl + Home`) |
| **Hydraulic Failure** | 1. **Slow Down**<br>2. **Gear** — Emergency Extend (if needed)<br>3. **Land** — Flapless (add 20-30 km/h) |
| **Spin Recovery** | 1. **Controls** — NEUTRAL<br>2. **Rudder** — FULL OPPOSITE<br>3. **Stick** — EASE FORWARD |
| **Ejection** | 1. **Eject** — INITIATE (`LCtrl + E` x3) |

## Realism Notes
- **Cockpit:** The DCS Su-25 (FC3) has a non-clickable cockpit.
- **Laser System:** The **Klen-PS** laser is fixed in the nose with limited gimbal. You cannot fly level and lase a target below you; you must maintain a dive (15-30°) towards the target until impact.
- **Navigation:** Uses RSBN (Russian TACAN) and PRMG (Russian ILS), simplified in FC3 but functional.

## Startup Checklist

### Pre-Start
- [ ] **Parking Brake** — SET (`W` key)
- [ ] **Throttle** — IDLE (full aft / `PgDn`)
- [ ] **Master Power** — ON (`RShift + L`)
- [ ] **Battery** — ON

### Engine Start
- [ ] **Left Engine** — START (`RAlt + Home`)
  - N2 rising, EGT rising — normal
  - N2 > 50% — advance throttle
- [ ] **Right Engine** — START (`RCtrl + Home`)
  - Wait for N2 > 50%
- [ ] Both engines at ~65% RPM IDLE — confirm
- [ ] **Generators** — ON (both)
- [ ] **Hydraulics** — check green
- [ ] **Fuel** — check quantity

### Avionics Power-On
- [ ] **R-862 Radio** — ON, set frequency
- [ ] **SPO-15 RWR** — ON
- [ ] **Weapons systems** — power ON

## Taxi & Takeoff

### Taxi
- [ ] **Parking Brake** — RELEASE (`W`)
- [ ] **Throttle** — advance slowly
- [ ] Taxi speed < 20 kph in turns

### Takeoff
- [ ] **Flaps** — TAKEOFF (`F`)
- [ ] **Throttle** — FULL (`PgUp`)
- [ ] Rotate at ~220 kph
- [ ] **Gear** — UP (`G`) at positive climb rate
- [ ] **Flaps** — RETRACT above 300 kph (`F`)

## Navigation

### Basic Navigation
The Su-25 uses a simplified navigation system with limited waypoint capability.

- [ ] **Reference heading** — set on compass
- [ ] **RSBN TACAN-equivalent** — if available in mission
- [ ] **Visual navigation** — primary method for low-level attack

### Navigation Tips
- Use F10 map (`F10`) for situational awareness
- Note terrain features and reference landmarks
- Pre-plan ingress/egress routes before takeoff

## Weapons Employment

### Weapons Available
| Weapon | Type | Notes |
|---|---|---|
| GSh-30-2 | 30mm cannon | Built-in, 250 rounds |
| S-5 pods | Rockets (57mm) | Area suppression |
| S-8 pods | Rockets (80mm) | Point targets |
| S-24 | Large unguided rocket | Bunkers, hard targets |
| OFAB-100/250 | Unguided bombs | General purpose |
| FAB-500 | 500kg bomb | Heavy targets |
| RBK-250/500 | Cluster bombs | Area denial |
| Kh-23M | Radio-guided missile | Manual command guidance |
| R-60 | IR AAM | Self-defense |

### Master Arm & Safety
- [ ] **Master Arm** — SAFE during all non-attack phases
- [ ] **Master Arm** — ARM (`O` key) only in attack phase

### Air-to-Ground Rockets

#### S-5 / S-8 Attack Profile
1. [ ] Select rocket pods on weapon panel
2. [ ] Set salvo size (single / pair / ripple)
3. [ ] Dive: 20°–35° toward target
4. [ ] Aim HUD pipper on target
5. [ ] Fire at 1,500–2,000 m range: `Space` or trigger
6. [ ] Pull off at 800 m minimum

#### S-24 Attack Profile
1. [ ] Select S-24 on weapon panel
2. [ ] Dive: 25°–35° dive angle
3. [ ] Release at 1,200–1,800 m range
4. [ ] Single rocket — very effective against bunkers

### Unguided Bombs

#### Level Bombing (CCIP with basic HUD)
1. [ ] **Master Arm** — ARM
2. [ ] Select bomb type on weapon panel
3. [ ] Level flight at medium altitude (1,500–2,500 m)
4. [ ] Use HUD bomb fall line/pip
5. [ ] Release: `Space`
6. [ ] Minimum: 600 m AGL (high-drag) / 1,500 m AGL (low-drag)

#### Dive Bombing
1. [ ] Enter dive 20°–45°
2. [ ] Align pipper on target
3. [ ] Release and break off
4. [ ] Pull off with at least 500 m clearance

### GSh-30-2 30mm Cannon

#### Strafing Run
1. [ ] Select gun on weapon panel
2. [ ] Dive: 10°–20° toward target
3. [ ] Open fire at 1,200 m, cease at 500 m
4. [ ] Burst length: 0.5–1.5 seconds (30–50 rounds)
5. [ ] Break off and check for damage

### Kh-23M Radio-Guided Missile
1. [ ] Select Kh-23M on weapon panel
2. [ ] Arm missile system
3. [ ] Launch at target area, 4–10 km range
4. [ ] **Manual guidance** — use joystick hat or keyboard to guide
5. [ ] Fly missile onto target visually
6. [ ] This is a challenging weapon requiring practice

## Defensive Systems

### SPO-15 Beryoza RWR
- Provides audio/visual warning of radar threats
- Direction indicator shows threat bearing

### Chaff & Flares
- [ ] **Chaff** — `[` key
- [ ] **Flares** — `]` key
- [ ] Dispense on missile launch warning
- [ ] Standard break: hard turn into threat + flares

### Defensive Sequence
1. Hear RWR spike/launch warning
2. **Break** hard turn (3–5G) toward threat bearing
3. **Flares** — 2–3 bursts (`]`)
4. **Chaff** — 1–2 bursts (`[`) if radar-guided
5. Dive toward terrain for masking

## Landing Checklist

- [ ] **Fuel** — check (> 400 kg)
- [ ] **Gear** — DOWN (`G`) — 3 green lights
- [ ] **Flaps** — LANDING (`F`)
- [ ] **Approach speed** — 270–290 kph
- [ ] **Touchdown speed** — ~230 kph
- [ ] **Brakes** — `W`
- [ ] **Drag Chute** — `C`

## Shutdown Checklist

- [ ] **Park** in designated spot
- [ ] **Parking Brake** — SET (`W`)
- [ ] **Throttle** — IDLE then CUT-OFF
- [ ] Wait for engine spool-down
- [ ] **Avionics** — OFF
- [ ] **Battery** — OFF
- [ ] **Master Power** — OFF

---

## Emergency Procedures

### Engine Fire
1. Throttle affected engine — IDLE then OFF
2. Pull fire handle
3. Proceed single-engine approach

### Hydraulic Failure
1. Emergency gear extension — `LCtrl + G`
2. Flapless landing — add 20–30 kph to approach

### Ejection
- `LCtrl + E` — hold 1 second
- Minimum safe altitude: ~100 m AGL
