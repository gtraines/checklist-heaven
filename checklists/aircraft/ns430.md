## Overview

The NS430 is a GPS navigation system based on the Garmin GNS 430, integrated as a clickable module in several DCS aircraft. It provides:

- GPS-based navigation
- Waypoint and flight plan management
- Airport/airfield database
- VOR/ILS frequency management
- CDI (Course Deviation Indicator) output
- Moving map display

---

## Compatible Aircraft

The NS430 module can be installed in:
- Ka-50 Black Shark (selected aircraft variants)
- Mi-8MTV2 Magnificent Eight
- UH-1H Huey
- L-39 Albatros
- SA342 Gazelle
- Yak-52
- Various other props and light jets

> Check mission editor for NS430 availability for each aircraft.

## NS430 Interface Layout

```
┌────────────────────────────────────────┐
│  [COM1 freq]        [NAV1 freq]        │
│  [COM2 freq]        [NAV2 freq]        │
│  ┌──────────────────────────────┐      │
│  │   Main Display Area          │      │
│  │   (Navigation / Map / Freq)  │      │
│  └──────────────────────────────┘      │
│  [D→] [MSG] [FPL] [PROC] [ENT]        │
│  [CLR] [CRSR] [Large Knob] [Small Knob]│
└────────────────────────────────────────┘
```

| Control | Function |
|---|---|
| Large outer knob | Select group/page |
| Small inner knob | Select item within page |
| CRSR button | Activate cursor for entry |
| ENT button | Confirm selection |
| CLR button | Clear / cancel |
| D→ (Direct-To) | Initiate direct-to navigation |
| MSG button | View messages |
| FPL button | Open flight plan page |
| PROC button | Approach procedures |

## Power-On & Initialization

1. [ ] **NS430 circuit breaker / switch** — ON (varies by aircraft)
2. [ ] NS430 display lights up — **splash screen**
3. [ ] GPS initialization in progress
   - "Searching for satellites..." — wait 1–3 minutes
   - "GPS OK" — navigation ready
4. [ ] **Database** — check current (warning if expired)
5. [ ] **Position** — confirm GPS position is correct
6. [ ] **COM/NAV frequencies** — set as needed

> **Note:** If indoors (hangar) or heavy cloud cover, GPS may take longer to acquire.

## Navigation Pages

### Navigating Between Page Groups

- **Large knob** rotates through page groups: NAV, WPT, AUX, NRST
- **Small knob** rotates through pages within a group
- Pages: NAV1, NAV2, NAV3, NAV4, MAP, NRST, FPL

---

### Default Navigation Page

**NAV1 Page** — primary navigation display:
```
BRG   DIS   ETE
↑270° 12.3NM 0:08

DTK   TRK   GS
265°  267°  142KT
```

| Field | Meaning |
|---|---|
| BRG | Bearing to active waypoint |
| DIS | Distance to active waypoint |
| ETE | Estimated time en route |
| DTK | Desired track |
| TRK | Actual track over ground |
| GS | Ground speed |

---

### Map Page

**MAP Page** — shows moving map:

- Own aircraft position in center
- Waypoints / flight plan shown
- Scale adjustable with inner knob
- Overlay: airways, airports, VORs

**Zoom:** rotate small inner knob when on MAP page

---

### Nearest Page

**NRST Group** — shows nearest airports, VORs, NDBs:

1. Rotate large knob to `NRST`
2. View list of nearest airports with distance/bearing
3. Press `ENT` to select for Direct-To navigation

---

### Flight Plan Page

**FPL Page** — shows active flight plan:

1. Press `FPL` button
2. View ordered waypoints
3. Active leg shown highlighted

## Waypoint Entry

### Enter a Waypoint by Identifier (ICAO code)

1. [ ] Press `D→` (Direct-To) OR navigate to FPL page
2. [ ] Press `CRSR` to activate cursor
3. [ ] **Rotate small knob** to enter first letter
4. [ ] **Rotate large knob** to move to next letter
5. [ ] Enter full identifier (e.g., `UGKO` for Kobuleti)
6. [ ] Press `ENT` to confirm

### Enter Waypoint by Coordinates

1. Navigate to WPT group → User Waypoints
2. Press `CRSR`
3. Select "NEW WAYPOINT"
4. Enter name
5. Enter Lat/Lon coordinates
6. Press `ENT`

---

## Flight Plan Creation

1. [ ] Press `FPL` button
2. [ ] Press `CRSR`
3. [ ] Cursor appears at first empty leg
4. [ ] **Enter departure** waypoint identifier → `ENT`
5. [ ] **Enter subsequent waypoints** → `ENT` for each
6. [ ] Continue until complete flight plan entered
7. [ ] Press `CRSR` to exit edit mode
8. [ ] Flight plan activated automatically

### Editing an Existing Flight Plan

1. Press `FPL`
2. Press `CRSR`
3. Move cursor to waypoint to edit
4. **Delete:** press `CLR`
5. **Insert before:** press `ENT` at cursor position
6. **Activate leg:** select leg, press `ENT`

---

## Direct-To Navigation

Go directly to any waypoint immediately.

1. [ ] Press `D→` button
2. [ ] Enter waypoint identifier (keyboard via knobs)
3. [ ] Press `ENT` once to confirm identifier
4. [ ] Press `ENT` again to activate
5. [ ] CDI/HSI now shows direct bearing to selected waypoint

> **Tip:** Quickest way to navigate to an airfield — D→ + ICAO code.

## Approach & ILS

### Loading an Instrument Approach

1. [ ] Press `PROC` button
2. [ ] Select `SELECT APPROACH`
3. [ ] Select airport (or use active destination)
4. [ ] Select approach type (ILS, VOR, RNAV, NDB)
5. [ ] Select runway
6. [ ] Confirm transition or vectors
7. [ ] `ENT` to load approach into flight plan

### ILS Approach with NS430
1. NS430 loads ILS approach waypoints
2. ILS frequency **auto-tuned** on NAV1 when inbound
3. CDI switches to ILS mode automatically on final
4. Fly CDI needles (lateral and glide slope) to touchdown

## Radio Management

### COM Radio Management

The NS430 controls COM1 and COM2 radios for compatible aircraft.

#### Setting COM1 Frequency
1. [ ] The **active frequency** is shown top-left (large number)
2. [ ] The **standby frequency** is shown smaller below
3. [ ] **Rotate inner knob** (bottom-right) to change standby MHz
4. [ ] **Rotate outer knob** to change kHz
5. [ ] **Press inner knob** to flip: standby becomes active
6. [ ] "Flip-Flop" — active ↔ standby swap

#### Preset Frequencies for CAS Operations

| Role | Typical Frequency |
|---|---|
| Guard (UHF) | 243.000 MHz |
| Guard (VHF) | 121.500 MHz |
| AWACS (mission) | Per briefing |
| ATC Tower | Per airport ATIS |
| FAC/JTAC | Per mission |

---

### NAV Radio Management

Same procedure as COM, but for NAV1/NAV2 (VOR/ILS).

1. Set NAV1 standby to ILS/VOR frequency
2. Press inner knob to flip active
3. CDI now shows navigation data from NAV1

## CDI / HSI Integration

The NS430 outputs CDI data to the aircraft's navigation instruments.

### CDI Modes
| Mode | Source | Use |
|---|---|---|
| GPS | NS430 GPS | Waypoint navigation |
| VLOC | NAV1 radio | VOR or ILS |

### Switching CDI Source
- Press `CDI` button on NS430 (if present)
- Or select on NAV page

### HSI Reading
- **Course arrow** shows selected course
- **Deviation bar** shows L/R correction needed
- **TO/FROM flag** shows if flying toward or away

## NS430 in Helicopters

The NS430 is particularly useful in helicopters which have limited native navigation:

### UH-1H Huey + NS430
- Provides GPS navigation in an otherwise VFR-only aircraft
- Set waypoints for LZ, FARP, rally points
- Monitor ETE and distance to LZ
- COM management for multi-frequency operations

### Mi-8 + NS430
- Primary navigation augmentation
- Flight plan filing and following

### Recommended Waypoints to Pre-Enter
| Waypoint Name | Description |
|---|---|
| HOME | Home base / main airfield |
| FARP | Forward Arming and Refueling Point |
| LZ1–4 | Landing Zones |
| IP1–2 | Initial Points for attack runs |
| RALLY | Emergency rally point |

## Tips & Tricks

### Fast Frequency Entry
- Use knobs efficiently: large = MHz integer, small = decimal
- Practice getting to 121.500 (Guard) quickly

### Nearest Airport in Emergency
1. Press `NRST`
2. First airport listed is nearest
3. Press `ENT` → `D→` to navigate directly

### NS430 as Moving Map
- Leave on MAP page during transit
- Provides situation awareness without F10 map
- Zoom to see ahead on flight plan

### Verify GPS Before Takeoff
- Confirm GPS OK status
- Verify own position matches actual location
- Cross-check with other navigation aids if available

## Keyboard & Mouse Reference

### NS430 Controls (all mouse-operated in DCS)
| Control | Action | Mouse |
|---|---|---|
| Large outer knob | Page group select | Left click + drag, or scroll |
| Small inner knob | Page select/frequency | Left click + drag, or scroll |
| CRSR | Cursor on/off | Left click |
| ENT | Enter/confirm | Left click |
| CLR | Clear/back | Left click |
| D→ | Direct-To | Left click |
| FPL | Flight plan | Left click |
| PROC | Procedures | Left click |
| MSG | Messages | Left click |

> In DCS, right-click and drag to rotate knobs, or scroll the mouse wheel. Specific binding depends on DCS version and aircraft.
