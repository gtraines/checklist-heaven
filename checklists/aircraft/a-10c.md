## Cockpit Orientation

```
┌──────────────────────────────────────────────────────┐
│  HUD               UFC (Upfront Controller)          │
│  Left MFCD         Right MFCD                        │
│  CDU (right)       AHCP (Armament)    CMSC (Counter) │
│  IFF               EAC (Arm/Consent)                 │
│  Left DDI          Right DDI (Engine Instruments)    │
└──────────────────────────────────────────────────────┘
```

| Panel | Location | Function |
|---|---|---|
| HUD | Forward | Primary flight/weapon display |
| Left MFCD | Center-left | TGP, navigation, weapon pages |
| Right MFCD | Center-right | Radar, system pages |
| UFC | Glareshield | Upfront controller for radios/nav |
| CDU | Right console | Navigation computer |
| AHCP | Left glare | Armament panel, Master Arm |
| CMSC | Right of MFCD | Chaff/Flare control |
| HMCS | Helmet | Helmet Mounted Cueing System |

## Startup Checklist

### Pre-Start
- [ ] **Parking Brake** — SET (`W`)
- [ ] **Throttle** — IDLE (full back)
- [ ] **Battery** — ON (cockpit left console)
- [ ] **Emergency Flight Controls** — check NORMAL

### Electrical Power
- [ ] **Battery Bus** — check
- [ ] **External Power** — connect if available
- [ ] **APU** (Ground Power) — ON

### Engine Start (TF34-GE-100A × 2)
- [ ] **Left Engine Start** — `RAlt + Home`
  - N1 rising through 15% — open throttle to IDLE
  - Fuel flow, oil pressure, EGT — monitor
  - N1 > 55%, EGT < 700°C — normal
- [ ] **Right Engine Start** — `RCtrl + Home`
  - Same procedure as left
- [ ] Both at IDLE (65–70% N1) — stable
- [ ] **Generators** — Left and Right ON
- [ ] **APU** — OFF after generators stable

### Avionics Startup
- [ ] **SAS** (Stability Aug System) — ON
- [ ] **EAC** (Engage Autopilot Consent) — ARM
- [ ] **CDU power** — ON (right console)
  - CDU self-test completes (~30 sec)
- [ ] **EGI** (Embedded GPS/INS) — ON
  - GPS alignment: wait for "GPS OK" (~3–5 min)
- [ ] **UFC** — ON (displays frequency)
- [ ] **MFCD Left/Right** — ON
- [ ] **TGP** — ON (power up early; takes 3–5 min to cool)
- [ ] **HMCS** — ON, boresight (if used)
- [ ] **IFF** — set mode/codes
- [ ] **Radios** — set frequencies (UFC or CDU)
  - VHF: ARC-186
  - UHF: ARC-164

### AHCP (Armament Handoff Control Panel)
- [ ] **Master Arm** — SAFE (until attack)
- [ ] **Laser Arm** — OFF (until attack)
- [ ] **TGP Power** — STBY then OPER
- [ ] **IFFCC** — TEST (check weapon computer)

## Taxi & Takeoff

### Before Taxi
- [ ] **Flight controls** — full sweep, check free
- [ ] **Trim** — set for takeoff
- [ ] **Flaps** — up (no flap required for standard TO)

### Taxi
- [ ] **Parking Brake** — RELEASE (`W`)
- [ ] Advance throttle slightly; A-10 moves slowly
- [ ] Use differential thrust and nosewheel for turns

### Takeoff
- [ ] **Master Arm** — SAFE until airborne
- [ ] **Throttle** — FULL (`PgUp`)
- [ ] Rotate at ~140 kts
- [ ] **Gear** — UP (`G`)
- [ ] Climb at 200 kts, 15° pitch

## Navigation (CDU/EGI)

### CDU Waypoint Setup
1. [ ] **CDU power** — ON
2. [ ] Press `WPT` on CDU to access waypoints
3. [ ] Enter waypoint coordinates (Lat/Lon or MGRS)
4. [ ] **STEER** — select waypoint for active navigation
5. [ ] **HSI** — shows bearing to active waypoint

### Navigation Display (HUD/MFCD)
| Display | Meaning |
|---|---|
| Steerpoint triangle | Heading to active waypoint |
| Distance readout | NM to steerpoint |
| Time readout | ETA to steerpoint |

### Changing Waypoints
- **CDU:** press WPT then increment/decrement
- **HOTAS:** DMS Up/Down (if programmed)

### ILS Approach
- [ ] **ILS frequency** — set on CDU/UFC
- [ ] **Approach course** — check
- [ ] **EGI/HSI** — ILS mode selected
- [ ] Intercept glide slope at 3,000 ft, 8 NM
- [ ] Fly ILS needles centered to touchdown

## Avionics & Systems

### MFCD Operations

#### MFCD Page Navigation
- **OSB (Option Select Buttons)** — around edge of display
- **OSB 1** — HQ (page menu) when nothing else
- **Common Pages:**
  - `WPN` — Weapon page
  - `TGP` — Targeting Pod
  - `STAT` — System status
  - `HSD` — Horizontal Situation Display
  - `CDU` — CDU mirror

#### HSD (Horizontal Situation Display)
- Shows own aircraft position, waypoints, threat rings
- Zoom: OSB around edge
- Select to show flight plan

---

### TGP (Targeting Pod — Litening AT or Sniper)

#### Power/Warmup
- [ ] **TGP switch** — STBY (from AHCP or MFCD)
- [ ] Wait 3–5 minutes for IR detector to cool
- [ ] **TGP switch** — OPER when ready

#### TGP Operation
1. Select `TGP` page on MFCD
2. **Slew TGP** to target area (HOTAS TDC or keyboard)
3. **Laser** — ARM from AHCP
4. **Track mode** — select (AREA, POINT, INR)
   - POINT track: lock on target feature
   - AREA track: area correlation
5. **Lase** to designate target (HOTAS pinkie switch / `O` key)
6. **SPI** (Sensor Point of Interest) — set from TGP lock

#### TGP FLIR Polarity
- **White-Hot** — hot objects appear white
- **Black-Hot** — hot objects appear black (default thermal)
- Toggle: OSB on TGP page

#### TGP FOV
- **Wide FOV (WFOV)** — general search
- **Narrow FOV (NFOV)** — identification
- **Expand (EFOV)** — maximum zoom
- Toggle: `*` numpad / HOTAS TMS Up

### HMCS

- [ ] **HMCS power** — ON
- [ ] **Boresight** — align with HUD (on ground, look at alignment dot)
- Provides weapon cueing and target designation via head movement
- Used with AGM-65, cannon, and A/A missiles

## Weapons Employment

### Pre-Attack Checks
- [ ] **Master Arm** — ARM (`O` key from AHCP)
- [ ] **EAC** — ARM (consent switch)
- [ ] **Laser Arm** — ARM (AHCP)
- [ ] **TGP** — OPER and tracking target
- [ ] **Weapon** — selected on MFCD WPN page
- [ ] **Delivery mode** — selected (CCIP, CCRP, AUTO)

---

### GAU-8/A 30mm Cannon

#### Employment
1. [ ] **Gun** — select (HOTAS master mode to A/G)
2. [ ] **IFFCC** — check gun solution on HUD
3. [ ] Dive: 10°–25°
4. [ ] **Open fire** at 1,500 m range
5. [ ] Short bursts: 1.5 seconds max
6. [ ] Break at 600 m minimum

> **Tip:** Use 1–2 second bursts. One burst = ~30–40 rounds. Very effective against light armor/vehicles.

---

### AGM-65 Maverick

#### Seeker Variants
| Variant | Seeker | Range | Target |
|---|---|---|---|
| AGM-65D | IIR | 500m–8km | Armor, vehicles |
| AGM-65G | IIR large | 500m–8km | Hardened targets |
| AGM-65K | EO TV | 500m–8km | Daylight/CAS |

#### AGM-65 Procedure
1. [ ] Select AGM-65 on WPN page
2. [ ] **MAV power** — ON (allow 3–5 min for D/G models)
3. [ ] **SLAVE** — slew seeker to TGP SPI (if TGP active)
   - Or manually slew via HOTAS
4. [ ] Seeker crosshairs on target — **lock**: `Numpad Enter`
5. [ ] Tone change confirms lock
6. [ ] **Launch envelope:** 500 m – 8 km
7. [ ] **Consent/Fire:** trigger or pickle button
8. [ ] **Fire-and-forget** — break off safely

### JDAM (GBU-31/32/38)

| Weapon | Weight | Notes |
|---|---|---|
| GBU-38 | 500 lb | Mk-82 + JDAM kit |
| GBU-32 | 1,000 lb | Mk-83 + JDAM kit |
| GBU-31 | 2,000 lb | Mk-84 + JDAM kit |

#### JDAM GPS-Guided Procedure
1. [ ] **Master Arm** — ARM
2. [ ] Select JDAM on WPN page (`WPN` MFCD)
3. [ ] **TGP** — designate target (lase or SPI set)
4. [ ] Coordinates loaded to JDAM via EGI
5. [ ] **Delivery mode** — CCRP (recommended)
6. [ ] Fly computed release line (HUD cue)
7. [ ] **Release** at computed point: pickle button
8. [ ] No further action required — GPS-guided
9. [ ] Effective in all weather, day/night

> **Tip:** Can deliver multiple JDAMs at different targets on single pass if coordinates set.

---

### Paveway LGB (GBU-10/12)

| Weapon | Weight | Notes |
|---|---|---|
| GBU-12 | 500 lb | Mk-82 + Paveway II laser kit |
| GBU-10 | 2,000 lb | Mk-84 + Paveway II laser kit |

#### LGB Procedure
1. [ ] **Master Arm** — ARM
2. [ ] **Laser Arm** — ARM (AHCP)
3. [ ] Select GBU-12/10 on WPN page
4. [ ] **TGP** — track target in POINT or AREA mode
5. [ ] **Delivery:** CCRP or DTOS mode
6. [ ] Fly release line; release at computed point
7. [ ] **Lase immediately after release** — keep TGP on target
8. [ ] Laser on until impact
9. [ ] Do not break hard or lose sight of target before impact

> **Tip:** Can also use BUDDY LASE — another aircraft lases while you deliver.

---

### WCMD (CBU-103/104/105)

Wind-Corrected Munitions Dispenser — cluster munition with GPS/INS guided dispenser.

1. [ ] Select CBU-103/104/105 on WPN page
2. [ ] Set fuze altitude for optimal burst
3. [ ] **Delivery:** CCRP
4. [ ] Fly release cue and release
5. [ ] **Fire-and-forget** GPS corrected trajectory

---

### SDB (GBU-39 Small Diameter Bomb)

- 250 lb guided bomb with extended range glide
- Can carry 4 per BRU-61 rack (16 per aircraft)
- GPS-guided, effective against lightly armored/soft targets

1. [ ] Select GBU-39 on WPN page
2. [ ] Set target coordinates via TGP or CDU
3. [ ] **Delivery:** CCRP at medium altitude
4. [ ] Can engage 4 separate targets per pass

---

### Unguided Bombs & Rockets

#### Mk-82 / Mk-84 Dumb Bombs
- CCIP delivery (aim pipper at target and release)
- Or CCRP at pre-planned coordinates

#### 2.75-in Rockets / 5-in Zuni
1. Select rocket pod on WPN page
2. Dive: 15°–25°
3. Aim pipper on target
4. Fire at 1,200–1,800 m range

## Defensive Systems

### ALR-69/ALR-56M RWR
- Radar warning with visual and audio cues
- Shows threats in 360° azimuth display

### ALE-47 Chaff/Flare Dispenser
- [ ] **CMSP** — ON, program selected
- [ ] **Chaff** — `[`
- [ ] **Flares** — `]`
- [ ] **Auto mode** — enables automatic dispensing
- Standard defensive break: hard turn + 2 bursts flares + 1 burst chaff

### ALQ-131 / ALQ-184 ECM Pod
- [ ] **ECM Power** — ON before entering threat area
- Provides radar jamming

### DIRCM (if equipped on A-10C)
- Directional IR countermeasures
- Automatic — engages IR missiles

## Landing Checklist

- [ ] **TGP** — STBY (cool down)
- [ ] **Master Arm** — SAFE
- [ ] **Fuel** — check (> 600 lbs)
- [ ] **Gear** — DOWN (`G`) — 3 green
- [ ] **Flaps** — check (not required but helps)
- [ ] **Approach speed** — 150 kts (no flap); 135 kts (20° flap)
- [ ] **Touchdown** — 120 kts
- [ ] **Brakes** — `W`

---

## Shutdown Checklist

- [ ] **Parking Brake** — SET (`W`)
- [ ] **TGP** — OFF
- [ ] **ECM** — OFF
- [ ] **Avionics** — all OFF (CDU, MFCD, UFC)
- [ ] **Throttle** — IDLE then CUT-OFF
- [ ] Engine spool-down (N1 < 20%)
- [ ] **Battery** — OFF

---

## Emergency Procedures

### Single Engine
1. Dead engine — confirm (no N1, no EGT)
2. Throttle dead engine — OFF
3. Live engine — FULL as needed
4. Jettison heavy stores if needed — `LAlt + J`
5. Approach: add 15 kts

### Loss of All Electrics
1. Battery to essential bus
2. Gravity-drop gear if needed
3. Fly to nearest field

### Ejection
- `LCtrl + E` — hold 1 second
